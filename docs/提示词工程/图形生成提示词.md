# 图形生成提示词工程

## 架构原则

**图文分离策略**：
- 前端渲染：纯文本公式（KaTeX/MathJax）
- 后端渲染：复杂图形（LaTeX/TikZ、Python等）

## 前端公式渲染提示词

### 数学公式生成指令

```
【数学公式格式要求】
- 使用KaTeX语法，不要使用纯文本符号
- 分数：\frac{分子}{分母}
- 根号：\sqrt{内容} 或 \sqrt[n]{内容}
- 上标：x^{2}
- 下标：x_{1}
- 求和：\sum_{i=1}^{n}
- 积分：\int_{a}^{b}
- 希腊字母：\alpha, \beta, \pi, \theta
- 三角函数：\sin, \cos, \tan
- 矩阵：\begin{bmatrix} a & b \\ c & d \end{bmatrix}

【示例】
错误：x²/4 + √3
正确：\frac{x^2}{4} + \sqrt{3}
```

### 化学方程式生成指令

```
【化学方程式格式要求】
- 使用mhchem语法（KaTeX扩展）
- 化学式：\ce{H2O}
- 化学反应：\ce{2H2 + O2 -> 2H2O}
- 离子：\ce{Na+}, \ce{SO4^2-}
- 沉淀符号：\ce{AgCl v}
- 气体符号：\ce{H2 ^}
- 加热条件：\ce{CaCO3 ->[加热] CaO + CO2 ^}

【示例】
\ce{2KMnO4 ->[加热] K2MnO4 + MnO2 + O2 ^}
```

## 后端图形渲染提示词

### 图形描述生成规范

AI生成题目时，如果需要配图，必须在题目JSON中添加`figure`字段，包含详细的图形描述。

```json
{
  "figure": {
    "type": "geometry|circuit|molecule|coordinate|timeline|...",
    "subject": "数学|物理|化学|...",
    "description": "详细的图形描述",
    "params": {
      // 图形参数
    },
    "constraints": [
      // 约束条件
    ]
  }
}
```

### 数学图形描述模板

#### 几何图形
```
【几何图形描述要求】
- 明确图形类型：三角形、圆、矩形、平行四边形等
- 标注所有点的名称：A、B、C、D等
- 说明边的关系：AB=3cm, BC=4cm, ∠ABC=90°
- 特殊标记：直角标记、等边标记、平行标记

【示例】
"直角三角形ABC，其中∠B为直角，AB=3cm，BC=4cm，AC为斜边。在B点处标注直角符号。"
```

#### 函数图像
```
【函数图像描述要求】
- 坐标系范围：x轴[-5, 5]，y轴[-3, 3]
- 函数表达式：y = x^2
- 关键点：顶点(0,0)，与x轴交点等
- 特殊标注：对称轴、最值点

【示例】
"在直角坐标系中绘制二次函数y=x²-2x-3的图像，x轴范围[-2,4]，y轴范围[-5,2]。标注顶点(1,-4)和与x轴交点(-1,0)和(3,0)。"
```

#### 立体几何
```
【立体几何描述要求】
- 明确立体图形类型：长方体、圆柱、圆锥、球等
- 透视关系：正等轴测、斜二测等
- 尺寸标注：长宽高、半径等
- 虚实线：看不见的边用虚线

【示例】
"正方体ABCD-A'B'C'D'，边长为2。使用正等轴测投影。底面ABCD在前，顶面A'B'C'D'在后。后面的边A'D'、B'C'、C'D'用虚线表示。"
```

### 物理图形描述模板

#### 力学图
```
【力学图描述要求 - 严格约束】
- 物体必须在支撑面上，不能悬空或穿透
- 接触点必须在边界上，不能进入内部
- 力的方向必须符合物理规律：
  * 重力：竖直向下
  * 支持力：垂直于接触面向上
  * 摩擦力：平行于接触面
  * 拉力：沿绳子方向
- 所有力必须从作用点出发
- 力的长度表示大小关系

【示例】
"斜面倾角30°，长度为L。小木块质量m放置在斜面上，木块底面与斜面完全接触（不能悬空或陷入）。从木块重心画三个力：重力G竖直向下，支持力N垂直斜面向上，摩擦力f沿斜面向上。"
```

#### 电路图
```
【电路图描述要求 - 严格约束】
- 电路必须闭合，不能有断路
- 元件连接必须正确：
  * 电源正负极不能直接相连
  * 电流表串联
  * 电压表并联
- 导线连接必须在元件端点
- 使用标准电路符号

【示例】
"串联电路：电源E（6V）→开关S→电阻R1（10Ω）→电阻R2（20Ω）→回到电源负极。在R1两端并联电压表V1，在R2两端并联电压表V2。电路必须闭合。"
```

#### 光路图
```
【光路图描述要求】
- 光线用带箭头的直线表示
- 入射角、反射角、折射角必须符合定律
- 法线用虚线表示
- 镜面或界面用实线表示

【示例】
"平面镜反射：入射光线从左上方以30°入射角射向镜面O点，画出法线（垂直于镜面的虚线），反射光线以30°反射角从O点射出。入射角=反射角=30°。"
```

### 化学图形描述模板

#### 有机化学结构式
```
【有机化学结构式要求】
- 使用chemfig或RDKit生成
- 明确碳链结构
- 标注官能团位置
- 双键、三键、苯环等特殊结构

【示例】
"乙醇分子结构式：CH3-CH2-OH。碳链为2个碳原子，第二个碳原子连接羟基-OH。"
```

#### 化学实验装置图
```
【化学实验装置要求 - 严格约束】
- 装置连接必须合理，不能漏气
- 气体收集装置方向正确：
  * 排水法：导管伸入集气瓶底部
  * 向下排空气法：导管在集气瓶口
  * 向上排空气法：导管在集气瓶底
- 加热装置：酒精灯、试管、铁架台位置合理
- 冷凝装置：冷凝管方向正确（下进上出）

【示例】
"制取氧气装置：试管倾斜固定在铁架台上，试管口略向下倾斜。试管内放KMnO4固体。试管口塞单孔橡皮塞，导管一端插入橡皮塞，另一端伸入盛满水的集气瓶底部（排水法收集）。酒精灯在试管下方加热。"
```

### 生物图形描述模板

#### 细胞结构图
```
【细胞结构图要求】
- 明确细胞类型：动物细胞、植物细胞、细菌等
- 标注主要结构：细胞核、细胞质、细胞膜、细胞壁等
- 比例合理：细胞核约占细胞体积的10%

【示例】
"植物细胞结构图：外层是细胞壁（矩形），内层是细胞膜（紧贴细胞壁）。细胞质中有细胞核（圆形，位于中央偏左），液泡（大型，占细胞体积50%），叶绿体（椭圆形，分散在细胞质中）。"
```

#### 遗传图谱
```
【遗传图谱要求】
- 使用标准符号：♂（雄性）、♀（雌性）
- 世代关系清晰：P（亲代）、F1（子一代）、F2（子二代）
- 基因型标注：AA、Aa、aa
- 表型标注：显性性状、隐性性状

【示例】
"豌豆杂交遗传图谱：P代：高茎（AA）×矮茎（aa）。F1代：全部高茎（Aa）。F1自交：F2代：高茎（AA、Aa）：矮茎（aa）=3:1。"
```

### 地理图形描述模板

#### 等高线地形图
```
【等高线地形图要求】
- 等高线间距均匀
- 等高线不相交、不分叉
- 等高线闭合表示山顶或盆地
- 等高线密集表示坡度陡

【示例】
"等高线地形图：绘制5条等高线，从外到内高度分别为100m、200m、300m、400m、500m。等高线间距相等，形成同心圆，表示山顶。在300m等高线处标注陡崖符号（等高线重合）。"
```

### 历史图形描述模板

#### 历史时间轴
```
【历史时间轴要求】
- 时间轴水平或垂直
- 时间刻度均匀
- 重大事件标注清晰
- 时间跨度合理

【示例】
"中国近代史时间轴（1840-1949）：水平时间轴，从左到右。标注关键事件：1840年鸦片战争、1911年辛亥革命、1919年五四运动、1921年中共成立、1949年新中国成立。"
```

## 图形约束规范嵌入

在生成图形描述时，必须嵌入对应科目的图形约束规范。

```javascript
function generateFigurePrompt(subject, figureType, description) {
  const constraints = FIGURE_CONSTRAINTS[subject][figureType];

  return `
【图形生成任务】
科目：${subject}
图形类型：${figureType}
描述：${description}

【约束规范】
${constraints.map(c => `- ${c}`).join('\n')}

【输出要求】
生成完整的LaTeX/TikZ代码，确保：
1. 符合上述约束规范
2. 图形清晰、准确
3. 标注完整、规范
4. 可直接编译为SVG

【LaTeX代码】
\documentclass[tikz,border=2mm]{standalone}
\usepackage{tikz}
% 根据需要引入其他宏包
\begin{document}
\begin{tikzpicture}
% 在这里生成TikZ代码
\end{tikzpicture}
\end{document}
`;
}
```

## 错误处理和降级方案

如果后端LaTeX渲染失败，提供降级方案：

1. **简化图形**：尝试生成更简单的版本
2. **文字描述**：用文字描述代替图形
3. **占位符**：显示"[图：xxx]"占位符

```javascript
// 降级方案示例
if (latexRenderFailed) {
  return `<div class="figure-placeholder">
    <div class="figure-icon">📊</div>
    <div class="figure-desc">[图：${description}]</div>
  </div>`;
}
```

## 性能优化

1. **缓存机制**：相同描述的图形只渲染一次
2. **异步加载**：图形生成不阻塞题目文字显示
3. **批量渲染**：多个图形合并为一次LaTeX编译
4. **CDN分发**：生成的SVG文件通过CDN加速

## 质量检查清单

生成图形后，AI需要自我检查：

- [ ] 图形是否符合物理/化学/生物规律？
- [ ] 几何关系是否正确？
- [ ] 标注是否完整清晰？
- [ ] 是否有穿透、悬空等错误？
- [ ] 比例是否合理？
- [ ] 是否符合学科约束规范？

如果检查不通过，重新生成。
