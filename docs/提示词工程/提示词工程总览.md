# 提示词工程总览

## 架构原则

所有提示词模板存储在后端 `server/prompts/` 目录下，前端不暴露任何提示词内容。
前端仅发送用户选择的参数（版本、年级、科目、知识点、考点、难度、题型），后端组装完整prompt。

## 提示词分类

### 1. 试题生成主提示词 (`exam-generation.js`)
- 根据用户选择的参数动态组装
- 包含：角色设定、教材版本约束、难度控制、题型规范、格式要求

### 2. 难度层级提示词 (`difficulty-levels.js`)
- 10级难度的精确定义
- 按年级×科目矩阵细化
- 每级难度对应的认知层次（记忆→理解→应用→分析→评价→创造）

### 3. 图形生成提示词 (`figure-generation.js`)
- 前端公式渲染指令（KaTeX格式）
- 后端图形渲染指令（LaTeX/TikZ代码生成）
- 图形约束规范嵌入

### 4. 知识点AI生成提示词 (`topic-suggestion.js`)
- 根据用户输入的关键词生成知识点建议
- 限定在当前教材版本和年级范围内

### 5. 答案解析生成提示词 (`answer-generation.js`)
- 详细解题步骤
- 知识点关联
- 易错点提示

## 提示词变量系统

| 变量名 | 说明 | 示例 |
|--------|------|------|
| `{version}` | 教材版本 | 人教版 |
| `{grade}` | 年级 | 高中一年级 |
| `{subject}` | 科目 | 数学 |
| `{topic}` | 知识点 | 三角函数 |
| `{examPoint}` | 考点 | 正弦函数图像与性质 |
| `{difficulty}` | 难度等级(1-10) | 5 |
| `{difficultyDesc}` | 难度描述 | 中等难度 |
| `{examType}` | 考试类型 | 期末考试 |
| `{questionTypes}` | 题型配置JSON | {...} |
| `{figureConstraints}` | 图形约束规范 | ... |
