# 物理科目渲染策略文档

## 1. 学科渲染特征概述

物理是K-12阶段图形与公式并重的学科，涵盖力学、电学、光学、热学等多个领域。视觉呈现特点包括：
- **公式特征**：物理量符号、单位、矢量表示、希腊字母高频使用
- **图形需求**：受力分析图、电路图、光路图、运动轨迹图、波形图
- **排版痛点**：矢量箭头标注、电路元件符号、立体透视图、动态过程示意

## 2. 前端渲染范围（纯文本公式类）

### a. 必须走前端渲染的公式/符号类型

#### 初中阶段（8-9年级）
- **物理量与单位**：`$v = 10 \text{ m/s}$`、`$F = 5 \text{ N}$`、`$P = 100 \text{ W}$`
- **基本公式**：
  - 速度：`$v = \frac{s}{t}$`
  - 密度：`$\rho = \frac{m}{V}$`
  - 压强：`$p = \frac{F}{S}$`
  - 功率：`$P = \frac{W}{t}$`
  - 欧姆定律：`$I = \frac{U}{R}$`
  - 电功率：`$P = UI = I^2R = \frac{U^2}{R}$`
- **单位换算**：`$1 \text{ km} = 10^3 \text{ m}$`、`$1 \text{ kW·h} = 3.6 \times 10^6 \text{ J}$`

#### 高中阶段（10-12年级）
- **矢量表示**：`$\vec{F}$`、`$\vec{v}$`、`$\vec{a}$`、`$|\vec{F}| = 10 \text{ N}$`
- **力学公式**：
  - 牛顿第二定律：`$\vec{F} = m\vec{a}$`
  - 动能：`$E_k = \frac{1}{2}mv^2$`
  - 重力势能：`$E_p = mgh$`
  - 动量：`$\vec{p} = m\vec{v}$`
  - 动量守恒：`$m_1\vec{v_1} + m_2\vec{v_2} = m_1\vec{v_1'} + m_2\vec{v_2'}$`
- **电磁学公式**：
  - 库仑定律：`$F = k\frac{q_1q_2}{r^2}$`
  - 电场强度：`$E = \frac{F}{q}$`
  - 磁通量：`$\Phi = BS\cos\theta$`
  - 法拉第电磁感应：`$\mathcal{E} = -\frac{d\Phi}{dt}$`
  - 洛伦兹力：`$\vec{F} = q\vec{v} \times \vec{B}$`
- **波动光学**：
  - 波速公式：`$v = \lambda f$`
  - 折射定律：`$n_1\sin\theta_1 = n_2\sin\theta_2$`
  - 薄透镜公式：`$\frac{1}{f} = \frac{1}{u} + \frac{1}{v}$`
- **热学公式**：
  - 理想气体状态方程：`$pV = nRT$`
  - 热力学第一定律：`$\Delta U = Q + W$`
- **近代物理**：
  - 光子能量：`$E = h\nu = \frac{hc}{\lambda}$`
  - 质能方程：`$E = mc^2$`
  - 德布罗意波长：`$\lambda = \frac{h}{p}$`

### b. 标准LaTeX语法示例

```latex
% 矢量
\vec{F}, \overrightarrow{AB}

% 物理量单位（使用 \text 或 siunitx 宏包）
10 \text{ m/s}
\SI{10}{\meter\per\second}  % 使用 siunitx

% 希腊字母（物理常用）
\theta, \phi, \omega, \lambda, \mu, \rho

% 上下标
v_0, a_x, E_k, m_1

% 分数与根式
\frac{1}{2}mv^2, \sqrt{2gh}

% 微分与积分
\frac{d\Phi}{dt}, \int F dx

% 特殊符号
\times (叉乘), \cdot (点乘), \approx (约等于)
```

### c. 前端渲染性能评估

**推荐前端渲染引擎**：KaTeX（优先）

**性能特点**：
- 物理公式相对数学更简洁，KaTeX 渲染速度 < 5ms/公式
- 矢量符号、希腊字母均为标准LaTeX，无需额外宏包

**所需前端宏包**：
- 基础符号：KaTeX 核心库
- 单位格式化：建议使用 `\text{}` 包裹单位，或引入 siunitx 支持

## 3. 后端渲染范围（复杂图形类）

### a. K-12全学段图形类型穷举

#### 1. 力学图形

**受力分析图**：
- 物体受力示意图（重力、支持力、摩擦力、拉力等）
- 力的合成与分解
- 力矩示意图

**后端渲染技术栈**：TikZ
```latex
\begin{tikzpicture}
  % 物体
  \draw[fill=gray!30] (0,0) rectangle (2,1);
  % 重力
  \draw[->,thick,red] (1,0.5) -- (1,-1) node[below] {$G$};
  % 支持力
  \draw[->,thick,blue] (1,0.5) -- (1,2) node[above] {$N$};
  % 摩擦力
  \draw[->,thick,green] (1,0.5) -- (-0.5,0.5) node[left] {$f$};
\end{tikzpicture}
```

**运动轨迹图**：
- 自由落体、抛体运动轨迹
- 圆周运动示意图
- 简谐运动图像

**后端渲染技术栈**：TikZ + pgfplots

#### 2. 电路图

**电路元件**：
- 电源、电阻、电容、电感
- 开关、电流表、电压表
- 二极管、三极管（高中选修）

**后端渲染技术栈**：circuitikz（专业电路图宏包）
```latex
\begin{circuitikz}
  \draw (0,0) to[battery1, l=$E$] (0,2)
        to[R, l=$R_1$] (2,2)
        to[R, l=$R_2$] (2,0)
        to[short] (0,0);
  \draw (2,2) to[ammeter, l=$A$] (4,2)
        to[lamp, l=$L$] (4,0)
        to[short] (2,0);
\end{circuitikz}
```

#### 3. 光路图

**光学现象**：
- 光的反射（平面镜、球面镜）
- 光的折射（棱镜、透镜）
- 光的干涉与衍射图样

**后端渲染技术栈**：TikZ
```latex
\begin{tikzpicture}
  % 透镜

\end{tikzpicture}
```

## 4. 边界与特殊情况处理

### 边界情况1：公式中的矢量箭头
**场景**：题目中出现 `$\vec{F} = 10 \text{ N}$`

**决断**：
- **前端渲染**：矢量符号完全由LaTeX处理

**理由**：`\vec{}` 是标准LaTeX命令，KaTeX原生支持。

---

### 边界情况2：电路图中的物理量标注
**场景**：电路图上标注电压 `$U_1 = 6 \text{ V}$`

**决断**：
- **后端渲染**：电路图和标注一起生成

**理由**：标注位置依赖电路元件位置，需统一计算。

---

### 边界情况3：受力分析图的力标注
**场景**：物体上标注 `$F = 20 \text{ N}$`

**决断**：
- **后端渲染**：图形和力标注一起生成

**理由**：力的方向、作用点需要精确对应物体位置。

---

### 边界情况4：简单示意图（如"斜面"）
**场景**：题目中出现"如图所示的斜面"

**决断**：
- **后端渲染**：即使是简单几何图形，也建议后端生成

**理由**：保证物理图形的专业性和一致性。

---

### 边界情况5：波形图的数学表达式
**场景**：波形图上标注 `$y = A\sin(\omega t + \phi)$`

**决断**：
- **后端渲染**：图像和公式一起生成

**理由**：公式位置需与波形对应。

---

## 5. 架构决策总结

### 前端渲染职责
- 所有纯文本物理公式
- 题目文字中的行内公式
- 物理量、单位、符号

### 后端渲染职责
- 所有受力分析图
- 所有电路图
- 所有光路图
- 所有波形图、场线图
- 所有实验装置图

### 性能指标
- 前端渲染：单个公式 < 5ms
- 后端渲染：电路图 < 800ms，受力图 < 300ms

### 技术栈最终选型
- **前端**：KaTeX 0.16+
- **后端**：TeX Live + TikZ + circuitikz + pgfplots
- **图片格式**：SVG（优先）+ PNG（@2x）
