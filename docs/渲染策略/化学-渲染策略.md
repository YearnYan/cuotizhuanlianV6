# 化学科目渲染策略文档

## 1. 学科渲染特征概述

化学是K-12阶段符号体系最独特的学科，拥有专属的化学方程式、结构式、电子式等表示方法。视觉呈现特点包括：
- **化学方程式**：反应物、生成物、条件标注、配平系数
- **结构式**：有机化学结构式、电子式、Lewis结构
- **特殊符号**：上下箭头（气体/沉淀）、可逆反应符号、电荷标注
- **排版痛点**：化学方程式的条件标注（加热、催化剂）、有机结构式的键角、电子式的点标注

## 2. 前端渲染范围（纯文本公式类）

### a. 必须走前端渲染的公式/符号类型

#### 初中阶段（8-9年级）
- **化学式**：`$\ce{H2O}$`、`$\ce{CO2}$`、`$\ce{NaCl}$`、`$\ce{Ca(OH)2}$`
- **化学方程式**：
  - `$\ce{2H2 + O2 ->[\text{点燃}] 2H2O}$`
  - `$\ce{CaCO3 + 2HCl -> CaCl2 + H2O + CO2 ^}$`
  - `$\ce{Fe + CuSO4 -> FeSO4 + Cu}$`
- **离子符号**：`$\ce{Na+}$`、`$\ce{Cl-}$`、`$\ce{SO4^{2-}}$`、`$\ce{Fe^{3+}}$`
- **化合价标注**：`$\overset{+1}{\ce{Na}}\overset{-1}{\ce{Cl}}$`
- **相对原子质量**：`$A_r(\ce{O}) = 16$`

#### 高中阶段（10-12年级）
- **离子方程式**：
  - `$\ce{H+ + OH- -> H2O}$`
  - `$\ce{Ba^{2+} + SO4^{2-} -> BaSO4 v}$`
- **氧化还原反应**：
  - `$\ce{Fe^{2+} -> Fe^{3+} + e-}$`
  - `$\ce{MnO4- + 5Fe^{2+} + 8H+ -> Mn^{2+} + 5Fe^{3+} + 4H2O}$`
- **热化学方程式**：
  - `$\ce{C(s) + O2(g) -> CO2(g)} \quad \Delta H = -393.5 \text{ kJ/mol}$`
- **化学平衡常数**：
  - `$K = \frac{[\ce{C}]^c[\ce{D}]^d}{[\ce{A}]^a[\ce{B}]^b}$`
  - `$K_w = [\ce{H+}][\ce{OH-}] = 10^{-14}$`
- **电化学**：
  - `$\ce{Zn - 2e- -> Zn^{2+}}$`（负极）
  - `$\ce{Cu^{2+} + 2e- -> Cu}$`（正极）
- **pH计算**：`$\text{pH} = -\lg[\ce{H+}]$`
- **摩尔浓度**：`$c = \frac{n}{V}$`
- **阿伏伽德罗常数**：`$N_A = 6.022 \times 10^{23} \text{ mol}^{-1}$`
- **气体摩尔体积**：`$V_m = 22.4 \text{ L/mol}$`

### b. 标准LaTeX语法示例

```latex
% 化学式与方程式（使用 mhchem 宏包）
\ce{H2O}
\ce{2H2 + O2 -> 2H2O}
\ce{CO2 + H2O <=> H2CO3}  % 可逆反应
\ce{CaCO3 ->[\Delta] CaO + CO2 ^}  % 加热条件

% 离子
\ce{Na+}, \ce{SO4^{2-}}, \ce{Fe^{3+}}

% 电子转移
\ce{Fe^{2+} -> Fe^{3+} + e-}

% 热化学方程式
\ce{C(s) + O2(g) -> CO2(g)} \quad \Delta H = -393.5 \text{ kJ/mol}

% 化合价标注
\overset{+1}{\ce{Na}}\overset{-1}{\ce{Cl}}

% 化学平衡常数
K = \frac{[\ce{C}]^c[\ce{D}]^d}{[\ce{A}]^a[\ce{B}]^b}

% pH
\text{pH} = -\lg[\ce{H+}]
```

### c. 前端渲染性能评估

**推荐前端渲染引擎**：KaTeX + mhchem 扩展

**性能特点**：
- KaTeX 原生支持 mhchem 宏包，渲染化学方程式 < 10ms
- 化学方程式的箭头、上下标均可前端处理

**所需前端宏包**：
- **mhchem**（必需）：化学方程式核心渲染
- **amsmath**：数学公式辅助

**关键配置**：
```javascript
// KaTeX 配置
katex.render(formula, element, {
  macros: {
    "\\ce": "\\require{mhchem}\\ce"
  }
});
```

## 3. 后端渲染范围（复杂图形类）

### a. K-12全学段图形类型穷举

#### 1. 有机化学结构式

**图形类型**：
- 简单结构式：甲烷、乙烯、乙炔
- 苯环及其衍生物
- 官能团标注
- 同分异构体
- 高分子链段

**后端渲染技术栈**：chemfig（LaTeX宏包）
```latex
% 使用 chemfig
\chemfig{H-C(-[2]H)(-[6]H)-C(-[2]H)(-[6]H)-H}  % 乙烷
\chemfig{*6(------)}  % 苯环
\chemfig{CH_3-CH(-[2]OH)-CH_3}  % 2-丙醇
```

#### 2. 电子式与Lewis结构

**图形类型**：
- 共价键电子式
- 离子键电子式
- 孤对电子标注

**后端渲染技术栈**：chemfig + TikZ
```latex
% Lewis结构
\chemfig{\lewis{0:2:4:6:,O}}  % 氧原子电子式
```

#### 3. 晶体结构图

**图形类型**：
- NaCl晶体结构
- 金刚石结构
- 石墨层状结构
- 干冰分子晶体

**后端渲染技术栈**：TikZ + tikz-3dplot

#### 4. 实验装置图

**图形类型**：
- 蒸馏装置
- 气体制备装置
- 滴定装置
- 电解装置
- 原电池装置

**后端渲染技术栈**：TikZ

#### 5. 能量变化图

**图形类型**：
- 反应能量曲线（活化能图）
- 盖斯定律能量图
- 化学平衡移动图

**后端渲染技术栈**：pgfplots
```latex
\begin{tikzpicture}
\begin{axis}[
  xlabel={反应进程},
  ylabel={能量},
  axis lines=left
]
\addplot[blue, thick, smooth] coordinates {
  (0,2) (1,4) (2,1)
};

\end{tikzpicture}
```

#### 6. 复杂分子三维结构

**图形类型**：
- 蛋白质分子结构
- 复杂有机分子3D构型
- 手性分子对比

**后端渲染技术栈**：RDKit（Python化学信息学库）
```python

mol = Chem.MolFromSmiles('c1ccccc1')
img = Draw.MolToImage(mol, size=(300, 300))
img.save('benzene.png')
```

## 4. 边界与特殊情况处理

### 边界情况1：简单化学方程式 vs 复杂化学方程式
**场景**：`$\ce{2H2 + O2 -> 2H2O}$` vs 带有复杂条件标注的方程式

**决断**：
- **前端渲染**：所有化学方程式（包括条件标注）

**理由**：mhchem 宏包完全支持条件标注语法 `\ce{->[\Delta]}` `\ce{->[催化剂]}`。

---

### 边界情况2：简单结构简式 vs 完整结构式
**场景**：`CH₃-CH₂-OH`（结构简式）vs 完整的键线式

**决断**：
- **前端渲染**：结构简式（纯文本形式）`$\ce{CH3-CH2-OH}$`
- **后端渲染**：键线式、球棍模型、空间填充模型

**理由**：结构简式本质是文本，键线式需要精确的键角和空间布局。

---

### 边界情况3：化合价标注
**场景**：`$\overset{+1}{\ce{Na}}\overset{-1}{\ce{Cl}}$`

**决断**：
- **前端渲染**：使用 `\overset{}{}` 实现

**理由**：纯文本标注，KaTeX原生支持。

---

### 边界情况4：电子式
**场景**：共价键电子式（如 H:Cl:）

**决断**：
- **后端渲染**：使用 chemfig 的 `\lewis{}` 命令

**理由**：电子点的精确位置需要专业排版，前端难以保证。

---

### 边界情况5：实验装置中的化学方程式标注
**场景**：实验装置图旁标注反应方程式

**决断**：
- **混合渲染**：
  - 装置图 → 后端渲染
  - 方程式标注 → 后端渲染时一并嵌入

**理由**：标注位置依赖装置图布局。

---

## 5. 架构决策总结

### 前端渲染职责
- 所有化学方程式（含条件标注）
- 离子方程式
- 热化学方程式
- 化学计算公式
- 结构简式
- 化合价标注

### 后端渲染职责
- 有机化学结构式（键线式、球棍模型）
- 电子式与Lewis结构
- 晶体结构图
- 实验装置图
- 能量变化曲线
- 复杂分子3D结构

### 技术栈最终选型
- **前端**：KaTeX + mhchem 扩展
- **后端**：TeX Live + chemfig + TikZ + pgfplots + RDKit
- **图片格式**：SVG（结构式）+ PNG（3D分子、实验装置）
