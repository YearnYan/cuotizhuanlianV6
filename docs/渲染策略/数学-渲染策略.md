# 数学科目渲染策略文档

## 1. 学科渲染特征概述

数学是K-12阶段公式密度最高的学科，涵盖从小学算术到高中微积分的全部符号体系。视觉呈现特点包括：
- **高频公式符号**：分数、根式、幂次、求和符号、积分符号等
- **几何图形需求**：平面几何、立体几何、函数图像、统计图表
- **排版痛点**：多行公式对齐、矩阵排版、分段函数、复杂嵌套结构

## 2. 前端渲染范围（纯文本公式类）

### a. 必须走前端渲染的公式/符号类型

#### 小学阶段（1-6年级）
- **四则运算**：`$3 + 5 = 8$`、`$12 \times 4 = 48$`
- **分数**：`$\frac{3}{4}$`、`$\frac{a+b}{c-d}$`
- **小数与百分数**：`$0.75$`、`$25\%$`
- **简单方程**：`$x + 5 = 12$`

#### 初中阶段（7-9年级）
- **代数式**：`$3x^2 - 2x + 1$`、`$(a+b)^2 = a^2 + 2ab + b^2$`
- **根式**：`$\sqrt{2}$`、`$\sqrt[3]{27}$`、`$\sqrt{a^2 + b^2}$`
- **一元二次方程**：`$x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$`
- **不等式**：`$x > 5$`、`$-3 \leq x < 7$`
- **函数表达式**：`$y = kx + b$`、`$y = ax^2 + bx + c$`
- **三角函数基础**：`$\sin\theta$`、`$\cos\alpha$`、`$\tan 45°$`
- **集合符号**：`$\{1, 2, 3\}$`、`$A \cup B$`、`$A \cap B$`
- **概率统计**：`$P(A) = \frac{m}{n}$`、`$\bar{x} = \frac{1}{n}\sum_{i=1}^{n}x_i$`

#### 高中阶段（10-12年级）
- **指数对数**：`$e^x$`、`$\log_a b$`、`$\ln x$`、`$a^{\frac{m}{n}}$`
- **三角恒等式**：`$\sin^2\alpha + \cos^2\alpha = 1$`、`$\sin(\alpha \pm \beta)$`
- **向量**：`$\vec{a} = (x, y)$`、`$\vec{a} \cdot \vec{b}$`、`$|\vec{a}|$`
- **数列**：`$a_n = a_1 + (n-1)d$`、`$S_n = \frac{n(a_1+a_n)}{2}$`
- **极限**：`$\lim_{x \to \infty} f(x)$`、`$\lim_{n \to \infty} \frac{1}{n}$`
- **导数**：`$f'(x)$`、`$\frac{dy}{dx}$`、`$\frac{d}{dx}(x^n) = nx^{n-1}$`
- **积分**：`$\int f(x)dx$`、`$\int_a^b f(x)dx$`、`$\int_0^1 x^2 dx$`
- **矩阵**：`$\begin{pmatrix} a & b \\ c & d \end{pmatrix}$`、`$\det(A)$`
- **复数**：`$z = a + bi$`、`$|z|$`、`$\bar{z}$`
- **排列组合**：`$C_n^m = \frac{n!}{m!(n-m)!}$`、`$A_n^m = \frac{n!}{(n-m)!}$`

### b. 标准LaTeX语法示例

```latex
% 分数
\frac{分子}{分母}

% 根式
\sqrt{表达式}
\sqrt[n]{表达式}

% 上下标
x^{上标}
x_{下标}

% 求和与积分
\sum_{i=1}^{n} x_i
\int_{a}^{b} f(x) dx

% 极限
\lim_{x \to \infty} f(x)

% 三角函数
\sin x, \cos x, \tan x

% 希腊字母
\alpha, \beta, \theta, \pi

% 矩阵
\begin{pmatrix}
a & b \\
c & d
\end{pmatrix}

% 分段函数
f(x) = \begin{cases}
x^2, & x \geq 0 \\
-x, & x < 0
\end{cases}

% 向量
\vec{a}, \overrightarrow{AB}

% 集合
\{1, 2, 3\}, A \cup B, A \cap B
```

### c. 前端渲染性能评估

**推荐前端渲染引擎**：KaTeX（优先）或 MathJax 3.x

**性能特点**：
- **KaTeX**：渲染速度极快（~5ms/公式），无需服务器端支持，适合高频交互场景
- **MathJax 3.x**：功能更全面，支持复杂宏包，但渲染速度较慢（~50ms/公式）

**所需前端宏包**：
- 基础数学符号：KaTeX/MathJax 核心库已包含
- 特殊符号扩展：`amsmath`、`amssymb`（矩阵、特殊运算符）

**性能优化建议**：
- 对于单页超过50个公式的试卷，建议使用虚拟滚动
- 缓存已渲染的公式DOM结构
- 使用Web Worker进行公式预处理

## 3. 后端渲染范围（复杂图形类）

### a. K-12全学段图形类型穷举

#### 1. 平面几何图形（小学-高中）
**图形类型**：
- 基础图形：三角形、四边形、圆、多边形
- 几何关系：平行线、垂直线、角平分线、中线
- 几何变换：平移、旋转、对称、相似

**后端渲染技术栈**：
```latex
% 使用 TikZ
\begin{tikzpicture}
  \coordinate (A) at (0,0);
  \coordinate (B) at (4,0);
  \coordinate (C) at (2,3);
  \draw (A) -- (B) -- (C) -- cycle;
  \draw (A) node[below left] {$A$};
  \draw (B) node[below right] {$B$};
  \draw (C) node[above] {$C$};
\end{tikzpicture}
```

#### 2. 立体几何图形（初中-高中）
**图形类型**：
- 基本几何体：长方体、正方体、圆柱、圆锥、球体、棱锥、棱柱
- 三视图：主视图、左视图、俯视图
- 截面图：平面与立体的交线

**后端渲染技术栈**：
```latex
% 使用 TikZ + tikz-3dplot
\tdplotsetmaincoords{70}{110}
\begin{tikzpicture}[tdplot_main_coords]
  \draw[->] (0,0,0) -- (4,0,0) node[anchor=north east]{$x$};
  \draw[->] (0,0,0) -- (0,4,0) node[anchor=north west]{$y$};
  \draw[->] (0,0,0) -- (0,0,4) node[anchor=south]{$z$};
  \draw (0,0,0) -- (3,0,0) -- (3,3,0) -- (0,3,0) -- cycle;
  \draw (0,0,3) -- (3,0,3) -- (3,3,3) -- (0,3,3) -- cycle;
  \draw (0,0,0) -- (0,0,3);
  \draw (3,0,0) -- (3,0,3);
  \draw (3,3,0) -- (3,3,3);
  \draw (0,3,0) -- (0,3,3);
\end{tikzpicture}
```

#### 3. 函数图像（初中-高中）
**图形类型**：
- 一次函数：直线
- 二次函数：抛物线
- 反比例函数：双曲线
- 指数函数、对数函数
- 三角函数：正弦、余弦、正切曲线
- 分段函数图像

**后端渲染技术栈**：
```latex
% 使用 pgfplots
\begin{tikzpicture}
\begin{axis}[
  axis lines=middle,
  xlabel=$x$,
  ylabel=$y$,
  domain=-5:5,
  samples=100
]
\addplot[blue, thick] {x^2};
\addplot[red, thick] {2*x + 1};
\end{axis}
\end{tikzpicture}
```

#### 4. 统计图表（小学-高中）
**图形类型**：
- 条形图、柱状图
- 折线图
- 扇形图（饼图）
- 直方图
- 茎叶图
- 箱线图（盒须图）
- 散点图

**后端渲染技术栈**：
```latex
% 使用 pgfplots
\begin{tikzpicture}
\begin{axis}[
  ybar,
  symbolic x coords={A,B,C,D},
  xtick=data,
  ylabel={频数}
]
\addplot coordinates {(A,5) (B,8) (C,12) (D,6)};
\end{axis}
\end{tikzpicture}
```

#### 5. 数轴与坐标系（小学-高中）
**图形类型**：
- 数轴（标注点、区间）
- 平面直角坐标系
- 极坐标系
- 空间直角坐标系

**后端渲染技术栈**：TikZ

#### 6. 概率树状图（初中-高中）
**图形类型**：
- 树状图（列举法）
- 概率树

**后端渲染技术栈**：TikZ + forest 宏包

|---------|-----------|---------|

## 4. 边界与特殊情况处理

### 边界情况1：简单几何图形标注
**场景**：题目中出现"如图，在△ABC中，AB=5，BC=7"

**决断**：
- **前端渲染**：仅文字公式部分（`$\triangle ABC$`、`$AB=5$`）
- **后端渲染**：几何图形本身

**理由**：几何图形涉及精确的点位、线段、角度关系，前端难以保证数学严谨性。

---

### 边界情况2：函数图像上的点标注
**场景**：在抛物线 y=x² 上标注点 A(2,4)

**决断**：
- **后端渲染**：整个图像（包括曲线和点标注）一次性生成

**理由**：点的位置需要与函数曲线精确对应，分离渲染会导致对齐问题。

---

### 边界情况3：统计图表中的数值标签
**场景**：柱状图顶部显示具体数值

**决断**：
- **后端渲染**：图表和数值标签一起生成

**理由**：数值标签的位置依赖于柱体高度，需要统一计算。

---

### 边界情况4：复杂多行公式（如矩阵、分段函数）
**场景**：
```latex
f(x) = \begin{cases}
x^2, & x \geq 0 \\
-x, & x < 0
\end{cases}
```

**决断**：
- **前端渲染**：KaTeX/MathJax 完全支持

**理由**：虽然结构复杂，但仍属于纯文本公式范畴，前端渲染性能足够。

---

### 边界情况5：带图形的应用题
**场景**：题目文字中嵌入小型示意图（如"如图所示的长方形"）

**决断**：
- **混合渲染**：
  - 题目文字 + 行内公式 → 前端渲染
  - 示意图 → 后端渲染为独立图片，通过 `<img>` 标签插入

**实现方案**：
```html
<p>如图所示，长方形ABCD的面积为 $S = ab$</p>
<img src="/api/render/geometry/rect_abcd.png" alt="长方形示意图">
```

---

### 边界情况6：动态交互图形（超出K-12范畴，但需预留接口）
**场景**：可拖动的几何图形、可调节参数的函数图像

**决断**：
- **前端渲染**：使用 GeoGebra、Desmos 等专业工具的嵌入式API
- **不使用后端渲染**：静态图片无法满足交互需求

**架构建议**：预留前端动态渲染模块接口，但当前版本不实现。

---

## 5. 架构决策总结

### 前端渲染职责
- 所有纯文本公式（从简单算术到高等数学符号）
- 题目文字中的行内公式
- 选项中的数学表达式

### 后端渲染职责
- 所有几何图形（平面、立体）
- 所有函数图像
- 所有统计图表
- 数轴、坐标系
- 概率树状图

### 性能指标
- 前端渲染：单个公式 < 10ms（KaTeX）
- 后端渲染：单个图形 < 500ms（TikZ编译）
- 缓存策略：后端图形按内容哈希缓存，命中率目标 > 80%

### 技术栈最终选型
- **前端**：KaTeX 0.16+ （主力）+ MathJax 3.x（降级方案）
- **后端**：TeX Live 2023 + TikZ + pgfplots + tikz-3dplot
- **图片格式**：SVG（优先，可缩放）+ PNG（降级，@2x高清）
