# 地理科目渲染策略文档

## 1. 学科渲染特征概述

地理是K-12阶段空间可视化需求最强的学科，涵盖自然地理与人文地理两大领域。视觉呈现特点包括：
- **地图类**：区域地图、等高线图、气候分布图、洋流图
- **示意图**：地球运动、大气环流、水循环、地质构造
- **统计图**：气温降水图、人口金字塔、产业结构图
- **排版痛点**：地图的精确性要求、等值线的平滑绘制、经纬度标注

## 2. 前端渲染范围（纯文本公式类）

### a. 必须走前端渲染的公式/符号类型

#### 初中阶段（7-8年级）
- **经纬度表示**：`$23.5°N$`、`$120°E$`、`$66.5°S$`
- **比例尺**：`$1:50000$`、`$1 \text{ cm} = 500 \text{ m}$`
- **海拔高度**：`$h = 8848.86 \text{ m}$`
- **温度**：`$T = -20°C$`、`$\Delta T = 6°C/1000 \text{ m}$`

#### 高中阶段（10-12年级）
- **地球运动参数**：
  - 自转角速度：`$\omega = \frac{2\pi}{24} \text{ rad/h} = 15°/\text{h}$`
  - 线速度：`$v = \omega R \cos\varphi$`
  - 公转周期：`$T = 365.25 \text{ d}$`
- **气温递减率**：`$\frac{\Delta T}{\Delta h} = -0.6°C/100 \text{ m}$`
- **人口增长率**：`$r = \frac{\text{出生率} - \text{死亡率}}{1000}$`
- **时区计算**：`$\text{时区} = \frac{\text{经度}}{15°}$`
- **太阳高度角**：`$H = 90° - |\varphi - \delta|$`
- **昼长计算**：`$D = \frac{2}{15}\arccos(-\tan\varphi \cdot \tan\delta)$`
- **地转偏向力**：`$f = 2\omega v \sin\varphi$`

### b. 标准LaTeX语法示例

```latex
% 经纬度
23.5°N, 120°E

% 比例尺
1:50000

% 温度递减
\frac{\Delta T}{\Delta h} = -0.6°C/100 \text{ m}

% 太阳高度角
H = 90° - |\varphi - \delta|

% 角速度
\omega = \frac{2\pi}{24} \text{ rad/h}

% 希腊字母
\varphi (纬度), \delta (赤纬), \omega (角速度)
```

- 地理公式简单，KaTeX 渲染 < 3ms/公式

## 4. 边界与特殊情况处理

### 边界情况1：简单经纬度标注 vs 地图
**场景**：题目中出现"该地位于 $30°N$, $120°E$"

**决断**：
- **前端渲染**：经纬度数值
- **后端渲染**：如果需要在地图上标注位置

**理由**：纯数值用前端，空间定位用后端。

---

### 边界情况2：简单比例尺 vs 地图比例尺标注
**场景**：`$1:50000$` vs 地图上的图形比例尺

**决断**：
- **前端渲染**：数字比例尺
- **后端渲染**：线段比例尺（图形化的比例尺标注）

**理由**：线段比例尺需要精确的长度对应关系。

---

### 边界情况3：气温降水数据表 vs 气候图
**场景**：
- 表格形式的月均气温降水数据
- 气温降水柱状折线图

**决断**：
- **前端渲染**：数据表格（HTML table）
- **后端渲染**：气候图（柱状折线复合图）

**理由**：表格是结构化文本，图表需要坐标轴和图形元素。

---

### 边界情况4：简化示意图 vs 精确地图
**场景**：
- 简化的大气环流示意图
- 精确的世界气候分布图

**决断**：
- **后端渲染（TikZ）**：简化示意图
- **后端渲染（Python）**：精确地图

**理由**：精确地图需要GIS数据支持，TikZ难以处理真实地理坐标。

---

### 边界情况5：地球光照图的投影方式
**场景**：侧视图 vs 俯视图（极地投影）

**决断**：
- **后端渲染**：统一使用 TikZ + tikz-3dplot

**理由**：两种投影都需要3D坐标变换。

---

## 5. 架构决策总结

### 前端渲染职责
- 经纬度、温度等数值
- 比例尺（数字形式）
- 地理计算公式
- 数据表格

### 后端渲染职责
- 所有地图类图形
- 等值线图
- 气候图表
- 地球运动示意图
- 大气环流图
- 地质构造图
- 人文地理统计图

### 特殊说明
地理学科的地图渲染是最大挑战。建议采用双轨策略：
1. **简化示意图**：使用 TikZ 生成（适合概念性图形）
2. **精确地图**：使用 Python Cartopy/matplotlib 生成（适合真实地理数据）

### 技术栈最终选型
- **前端**：KaTeX
- **后端**：TeX Live + TikZ + pgfplots + tikz-3dplot + Python (Cartopy, matplotlib)
- **图片格式**：SVG（示意图）+ PNG（地图，@2x）
- **地图数据源**：Natural Earth 数据集
