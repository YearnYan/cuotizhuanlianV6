# 数学 - 题目图形约束规范

## 1. 图形类型清单

### 1.1 平面几何图形
- **基本图形**：点、线段、射线、直线、角
- **三角形**：等边、等腰、直角、锐角、钝角三角形
- **四边形**：正方形、长方形、平行四边形、梯形、菱形
- **圆与扇形**：圆、半圆、扇形、弓形、圆环
- **多边形**：正五边形、正六边形、任意多边形
- **组合图形**：由基本图形拼接或挖去部分形成的复合图形

### 1.2 立体几何图形
- **棱柱**：正三棱柱、正四棱柱（长方体/正方体）、正六棱柱
- **棱锥**：正三棱锥（正四面体）、正四棱锥
- **棱台**：棱台截面图
- **圆柱、圆锥、圆台、球**
- **组合体**：多个基本体的拼接或挖切
- **截面图**：立体图形的截面展示
- **展开图**：立体图形的表面展开

### 1.3 坐标系与函数图像
- **直角坐标系（笛卡尔坐标系）**：二维平面直角坐标系
- **极坐标系**：高中选修内容
- **数轴**：一维坐标轴
- **一次函数图像**：直线
- **二次函数图像**：抛物线
- **反比例函数图像**：双曲线
- **指数函数/对数函数图像**
- **三角函数图像**：正弦、余弦、正切曲线
- **分段函数图像**
- **不等式解集区域**：坐标系中的阴影区域

### 1.4 统计与概率图形
- **条形统计图/柱状图**
- **折线统计图**
- **扇形统计图/饼图**
- **频率分布直方图**
- **频率分布折线图**
- **茎叶图**
- **散点图**
- **回归直线图**
- **树形图（概率）**
- **维恩图（集合）**

### 1.5 数论与代数图形
- **数轴标注图**：整数、分数、无理数的位置
- **因数/倍数关系图**
- **集合关系图**：交集、并集、补集的韦恩图
- **数列图示**：等差/等比数列的图形表示

### 1.6 变换与作图
- **对称图形**：轴对称、中心对称
- **平移图形**
- **旋转图形**
- **相似图形**
- **尺规作图步骤图**

## 2. 物理可实现性约束

### 2.1 几何图形比例约束（核心约束）
- 所有几何图形的边长比例必须与题目数据一致
- 若题目给出 AB=3, BC=4，则图中 AB:BC 必须接近 3:4
- 角度大小必须与标注一致：标注为直角的必须渲染为90°
- 锐角必须明显小于90°，钝角必须明显大于90°
- 等边三角形三边必须等长，三角必须均为60°
- 正方形四边必须等长，四角必须为直角

### 2.2 坐标系约束（核心约束）
- X轴和Y轴必须严格垂直
- 刻度间距必须均匀（等距）
- 原点必须标注为O
- 轴的正方向必须标注箭头
- 刻度数值必须等差递增
- 坐标轴标签（x, y）必须位于轴的正方向端

### 2.3 函数图像约束（核心约束）
- 函数图像必须连续（除已知间断点外）
- 函数图像必须通过所有已标注的关键点
- 一次函数必须为直线
- 二次函数必须为光滑抛物线，对称轴正确
- 反比例函数必须位于正确的象限，且不与坐标轴相交
- 三角函数周期、振幅、相位必须正确
- 指数函数必须单调，且过点(0,1)
- 对数函数必须单调，且过点(1,0)
- 渐近线必须正确标注，图像不得穿过渐近线

### 2.4 立体图形约束
- 透视关系必须一致（统一使用斜二测画法或正交投影）
- 可见边用实线，不可见边用虚线
- 平行关系在投影中必须保持平行
- 正方体/长方体的对面必须平行且全等

## 3. 几何关系约束

### 3.1 点线面关系
- 共线点必须严格在同一直线上
- 共面点必须严格在同一平面上
- 垂直关系：两线段夹角必须渲染为精确的90°，并标注直角符号"⊥"
- 平行关系：两线段在图中必须严格平行，间距一致
- 相交关系：交点必须精确位于两线段的交汇处

### 3.2 角度关系
- 互补角之和必须为180°
- 互余角之和必须为90°
- 对顶角必须相等
- 三角形内角和必须为180°
- 多边形内角和必须为(n-2)×180°
- 圆心角与圆弧的对应关系必须正确

### 3.3 长度与面积关系
- 三角形任意两边之和必须大于第三边
- 勾股定理：直角三角形的三边必须满足 a²+b²=c²
- 相似图形的对应边必须成比例
- 全等图形的对应边必须相等

### 3.4 圆的约束
- 圆心到圆上任意点的距离必须相等（半径一致）
- 切线必须与半径垂直于切点
- 弦的垂直平分线必须过圆心
- 同弧上的圆周角必须相等
- 圆周角等于同弧圆心角的一半

### 3.5 坐标几何约束
- 点的坐标标注必须与其在坐标系中的位置一致
- 两点间距离必须符合距离公式
- 直线斜率必须与图像倾斜方向一致（正斜率向右上，负斜率向右下）

## 4. 标注规范

### 4.1 点的标注
- 使用大写英文字母：A, B, C, D...
- 字母位于点的右上方（默认），避免与线段重叠
- 特殊点标注：O（原点/圆心）、P（动点）、M/N（中点）
- 坐标标注格式：A(x, y)，括号内用逗号分隔

### 4.2 线段与角的标注
- 线段用两端点字母表示：AB、CD
- 线段长度标注在线段中部上方或下方
- 角度标注使用弧线标记，数值写在弧线旁
- 直角使用小正方形符号标记
- 平行标记使用">"或">>"箭头符号
- 等长标记使用短横线"|"或"||"

### 4.3 函数标注
- 函数表达式标注在图像旁，不与图像重叠
- 关键点坐标必须标注：顶点、交点、零点
- 渐近线使用虚线绘制，并标注方程
- 定义域和值域在需要时用虚线框标注

### 4.4 数学符号规范
- 使用标准数学字体（斜体表示变量，正体表示常数和函数名）
- 希腊字母：α, β, γ, θ 用于角度
- 运算符号：×, ÷, ±, ≤, ≥, ≠, ≈
- 集合符号：∈, ∉, ⊂, ⊃, ∩, ∪, ∅

## 5. 常见错误清单

| 错误类型 | 错误描述 | 修正方案 |
|---------|---------|---------|
| 比例失真 | 标注AB=3,BC=5但图中AB比BC还长 | 按实际数值比例渲染，或明确标注"图形不按比例" |
| 直角不直 | 标注为直角但渲染角度明显不是90° | 使用精确角度计算，强制90°渲染 |
| 坐标刻度不均 | X轴刻度间距不等 | 使用等分算法确保刻度均匀 |
| 函数图像断裂 | 连续函数图像出现断点 | 增加采样点密度，使用贝塞尔曲线平滑 |
| 抛物线不对称 | 二次函数图像左右不对称 | 以对称轴为基准镜像渲染 |
| 虚实线混淆 | 立体图形可见边用虚线 | 严格按遮挡关系区分虚实线 |
| 平行线不平行 | 标注平行但图中明显不平行 | 使用相同斜率渲染，像素级校验 |
| 圆不圆 | 圆形渲染为椭圆 | 使用标准圆弧算法，确保半径一致 |
| 切线不切 | 切线与圆相交而非相切 | 计算切点坐标，确保切线与半径垂直 |
| 交点偏移 | 两图像交点与标注坐标不一致 | 先计算交点坐标，再精确定位 |
| 渐近线穿越 | 函数图像穿过渐近线 | 在渐近线附近增加采样密度，限制图像范围 |
| 展开图折叠错误 | 展开图无法正确折叠为原立体 | 使用标准展开算法校验可折叠性 |
| 对称轴错误 | 轴对称图形的对称轴位置不正确 | 计算对称轴方程，精确渲染 |
| 向量方向错误 | 向量箭头方向与坐标不一致 | 根据起点终点坐标确定箭头方向 |
| 阴影区域错误 | 不等式解集区域标注错误 | 使用测试点法验证阴影区域正确性 |
