# 物理 - 题目图形约束规范

## 1. 图形类型清单

### 1.1 力学图形
- **受力分析图**：物体受力的矢量图（重力、弹力、摩擦力、拉力等）
- **力的合成与分解图**：平行四边形法则图示
- **物体运动场景图**：斜面滑块、传送带、弹簧振子、单摆
- **运动轨迹图**：抛体运动、圆周运动轨迹
- **v-t图/s-t图/a-t图**：运动学图像
- **弹簧模型图**：弹簧连接物体的示意图
- **滑轮组图**：定滑轮、动滑轮、滑轮组
- **杠杆图**：杠杆平衡示意图
- **碰撞模型图**：两物体碰撞前后状态

### 1.2 电学图形
- **电路图**：串联、并联、混联电路
- **电路元件符号**：电源、开关、电阻、电容、电感、电流表、电压表
- **U-I特性曲线**：欧姆定律图像
- **电场线图**：点电荷、平行板电容器电场线
- **等势线图**：电场中的等势面截面

### 1.3 磁学图形
- **磁场线图**：条形磁铁、通电螺线管磁场线
- **安培力/洛伦兹力示意图**：带电粒子在磁场中的运动
- **电磁感应图**：导体切割磁力线、磁通量变化示意
- **变压器原理图**：原副线圈示意

### 1.4 光学图形
- **光路图**：反射、折射光路
- **透镜成像图**：凸透镜、凹透镜成像光路
- **光的色散图**：三棱镜色散
- **干涉/衍射图样**：双缝干涉条纹
- **全反射光路图**

### 1.5 热学图形
- **p-V图/p-T图/V-T图**：气体状态变化图像
- **分子运动模型图**：分子间距与力的关系
- **热机工作循环图**：卡诺循环等

### 1.6 波动与振动图形
- **波形图**：横波、纵波的波形
- **振动图像**：简谐运动的x-t图
- **波的干涉/衍射图**
- **驻波图**
- **多普勒效应示意图**

### 1.7 近代物理图形
- **原子模型图**：玻尔模型、能级图
- **光电效应图**：光电效应实验装置及I-U曲线
- **核反应示意图**：α衰变、β衰变
- **粒子轨迹图**：云室中的粒子径迹

### 1.8 实验装置图
- **力学实验**：打点计时器、气垫导轨、弹簧测力计
- **电学实验**：伏安法测电阻、电桥、示波器
- **光学实验**：光具座、分光计
- **热学实验**：量热器

## 2. 物理可实现性约束（核心约束）

### 2.1 通用物理约束 —— 禁止穿透/穿模/漂浮/错层/反向接触/无支撑悬空

#### 2.1.1 接触约束
- **物体必须与支撑面边界接触**：物块底面必须与桌面/斜面表面重合，禁止悬浮于表面上方
- **禁止穿透**：物体不得穿入支撑面内部，物块底面不得低于桌面/斜面表面
- **禁止穿模**：两个物体不得相互重叠穿透，接触面必须是边界接触
- **接触面法线方向必须正确**：物体间的接触力方向必须垂直于接触面

#### 2.1.2 支撑约束
- **禁止无支撑悬空**：除明确标注"自由落体"或"抛体"外，物体必须有支撑
- **支撑关系必须连续**：A放在B上，B放在C上，则A-B-C必须依次接触
- **悬挂物体必须有绳/杆连接**：悬挂物体必须有可见的绳索或杆件连接到固定点

#### 2.1.3 层级约束
- **禁止错层**：叠放的物体必须按正确的上下顺序排列
- **禁止反向接触**：物体不能从支撑面的反面接触（如物块不能贴在桌面下方而不掉落，除非有明确的固定装置）

### 2.2 力学专项约束

#### 2.2.1 斜面与滑块约束
- 滑块/物块必须位于斜面表面上，底面与斜面完全贴合
- 滑块不得嵌入斜面内部（穿模）
- 滑块不得悬浮于斜面上方（漂浮）
- 斜面角度必须与题目标注一致
- 多个物块叠放时，上方物块底面必须与下方物块顶面贴合

#### 2.2.2 力的方向约束
- **重力**：必须竖直向下，作用点在重心
- **支持力（法向力）**：必须垂直于接触面，方向指向被支持物体
- **摩擦力**：必须沿接触面方向，与相对运动/运动趋势方向相反
- **弹力**：沿弹簧/绳索方向，指向恢复原长的方向
- **拉力**：沿绳索方向，绳索必须绷直（除明确标注松弛外）
- **浮力**：必须竖直向上

#### 2.2.3 绳索与轨道约束
- 绳索必须连续，不得出现断裂
- 绳索在无外力作用段必须为直线
- 绳索经过滑轮时必须与滑轮相切
- 轨道必须连续光滑（除明确标注粗糙段外）
- 圆形轨道必须为标准圆弧
- 物体在轨道上的位置必须在轨道表面上

#### 2.2.4 弹簧约束
- 弹簧两端必须分别连接物体或固定点
- 弹簧的自然长度、压缩长度、拉伸长度必须在图中有区分
- 弹簧的螺旋间距应均匀（除标注非线性弹簧外）

#### 2.2.5 滑轮约束
- 定滑轮轴心必须固定
- 动滑轮必须可随绳索移动
- 绳索必须绕过滑轮槽，不得脱离
- 滑轮组的绳索段数必须与题目一致

### 2.3 电学专项约束

#### 2.3.1 电路连接约束
- **电路必须闭合**：从电源正极出发，经过外电路，必须能回到电源负极
- **导线不得断开**：除开关断开状态外，导线必须连续
- **禁止短路**：导线不得直接连接电源正负极（除题目明确要求外）
- **元件连接必须正确**：电流表串联、电压表并联
- **电表量程必须合理**：电流表/电压表的量程必须能覆盖测量值

#### 2.3.2 电路元件符号约束
- 必须使用国标电路符号（GB/T 4728）
- 电源必须标注正负极
- 电阻、电容、电感符号必须规范
- 开关状态（断开/闭合）必须与题意一致
- 滑动变阻器必须标注滑片位置和接线方式

#### 2.3.3 电场与磁场约束
- 电场线必须从正电荷出发，终止于负电荷
- 电场线不得相交
- 电场线密度必须反映场强大小
- 等势线必须与电场线垂直
- 磁场线必须闭合（从N极到S极外部，S极到N极内部）
- 磁场线不得相交

### 2.4 光学专项约束

#### 2.4.1 光路约束
- **反射定律**：入射角必须等于反射角，入射光线、反射光线、法线在同一平面内
- **折射定律**：入射角与折射角必须满足 n₁sinθ₁ = n₂sinθ₂
- 光从光密介质到光疏介质，折射角 > 入射角
- 光从光疏介质到光密介质，折射角 < 入射角
- **全反射**：入射角大于临界角时，不得出现折射光线
- 法线必须垂直于界面
- 光路中的虚线表示法线或虚像光路的延长线

#### 2.4.2 透镜成像约束
- 凸透镜：平行光必须会聚于焦点
- 凹透镜：平行光必须发散，反向延长线过虚焦点
- 过光心的光线方向不变
- 成像位置必须符合透镜公式 1/u + 1/v = 1/f
- 实像用实线，虚像用虚线

### 2.5 热学约束
- p-V图中等温线必须为双曲线形状
- 等压过程在p-V图中必须为水平线
- 等容过程在p-V图中必须为竖直线
- 气体状态变化必须满足理想气体状态方程

### 2.6 波动约束
- 波形图中波长必须均匀（等间距的波峰/波谷）
- 横波的质点振动方向必须垂直于波的传播方向
- 纵波的质点振动方向必须平行于波的传播方向
- 驻波的波节位置必须固定不动

## 3. 几何关系约束

### 3.1 矢量图约束
- 力的矢量箭头长度必须与力的大小成正比
- 合力必须为分力的矢量和（平行四边形对角线）
- 力的作用点必须在物体上（重力在重心，支持力在接触点）
- 同一物体的所有力的作用点应标注在同一点（简化受力图）或各自正确位置（详细受力图）

### 3.2 运动学图像约束
- v-t图中匀速运动为水平线
- v-t图中匀加速运动为斜直线
- v-t图中图线下方面积等于位移
- s-t图中匀速运动为斜直线
- s-t图中静止为水平线
- 图像的斜率必须与物理量的变化率一致

### 3.3 电路拓扑约束
- 串联元件必须在同一条支路上
- 并联元件必须连接在同两个节点之间
- 电路的拓扑结构必须与题目描述一致
- 节点处电流必须满足基尔霍夫电流定律

### 3.4 光路几何约束
- 入射光线、法线、反射光线必须共面
- 入射角 = 反射角（以法线为基准测量）
- 折射光线必须在法线的另一侧
- 平行光经凸透镜后必须交于焦点

## 4. 标注规范

### 4.1 物理量标注
- 力的标注：使用矢量符号 F⃗，或加粗 **F**
- 速度标注：v，加速度：a，位移：s/x
- 电流：I，电压：U，电阻：R
- 标注格式：物理量符号 = 数值 + 单位（如 F = 10N）

### 4.2 方向标注
- 使用箭头表示矢量方向
- 重力方向标注"mg"或"G"，箭头竖直向下
- 速度方向标注"v"，箭头沿运动方向
- 电流方向用箭头标注在导线上

### 4.3 角度标注
- 斜面倾角用θ标注，弧线标在斜面与水平面之间
- 入射角/反射角/折射角用弧线标注，数值写在弧线旁
- 力与水平方向/竖直方向的夹角必须明确标注

### 4.4 电路标注
- 电源标注EMF（ε）和内阻（r）
- 电阻标注阻值（R₁, R₂...）
- 电表标注类型（A/V）和量程
- 开关标注编号（S₁, S₂...）

### 4.5 坐标轴标注
- 横轴和纵轴必须标注物理量名称和单位
- 格式：物理量/单位（如 v/(m·s⁻¹)）
- 刻度值必须标注在轴上

## 5. 常见错误清单

| 错误类型 | 错误描述 | 修正方案 |
|---------|---------|---------|
| **穿模** | 物块嵌入斜面内部 | 物块底面必须与斜面表面精确贴合 |
| **漂浮** | 物块悬浮在桌面上方 | 物块底面Y坐标必须等于桌面Y坐标 |
| **穿透** | 两个物体相互重叠 | 碰撞检测，确保物体边界不重叠 |
| **无支撑悬空** | 物体无任何支撑却不下落 | 必须有支撑面、绳索或杆件支撑 |
| **重力方向错误** | 重力不是竖直向下 | 重力矢量必须严格竖直向下 |
| **支持力方向错误** | 支持力不垂直于接触面 | 法向力必须垂直于接触面 |
| **摩擦力方向错误** | 摩擦力方向与运动方向相同 | 摩擦力必须与相对运动方向相反 |
| **电路不闭合** | 电路存在断点 | 检查从正极到负极的完整通路 |
| **电路短路** | 导线直接连接正负极 | 检查是否有元件被旁路 |
| **电表接法错误** | 电流表并联或电压表串联 | 电流表必须串联，电压表必须并联 |
| **反射角≠入射角** | 光路反射角度不正确 | 以法线为基准，确保两角相等 |
| **折射方向错误** | 光从密到疏介质折射角小于入射角 | 按折射定律计算正确角度 |
| **磁场线不闭合** | 磁场线有起点无终点 | 磁场线必须形成闭合回路 |
| **电场线相交** | 两条电场线在非电荷处相交 | 电场线不得相交 |
| **力的比例失调** | 10N的力箭头比5N的力还短 | 矢量箭头长度必须与大小成正比 |
| **弹簧断裂** | 弹簧两端未连接物体 | 弹簧必须两端有连接 |
| **绳索穿过物体** | 绳索路径穿过固体物体 | 绳索必须绕过物体表面 |
| **滑轮绳索脱离** | 绳索未贴合滑轮槽 | 绳索必须与滑轮相切 |
| **v-t图斜率错误** | 匀加速运动图线弯曲 | 匀加速运动v-t图必须为直线 |
| **波形不均匀** | 同一列波的波长不一致 | 波峰间距必须等于波长 |
