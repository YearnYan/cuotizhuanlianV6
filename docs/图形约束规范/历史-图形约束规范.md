# 历史 - 题目图形约束规范

## 1. 图形类型清单

### 1.1 时间线与年表
- **朝代更替时间轴**：中国历史朝代顺序
- **世界历史大事年表**：重大事件的时间线
- **并行时间轴**：中外历史对照时间线
- **历史分期图**：古代、近代、现代的划分

### 1.2 历史地图
- **疆域图**：各朝代/帝国的疆域范围
- **战争形势图**：战役进军路线、战场态势
- **迁徙/传播路线图**：民族迁徙、文化传播、贸易路线
- **殖民地分布图**：近代殖民扩张
- **革命根据地分布图**：中国近现代革命地图

### 1.3 政治制度图
- **政治体制结构图**：三省六部制、三权分立等
- **官僚体系层级图**：中央到地方的行政层级
- **选官制度演变图**：世卿世禄→察举→科举

### 1.4 经济与社会图形
- **经济发展数据图**：GDP、贸易额等统计图表
- **土地制度演变图**：井田制→均田制→租庸调等
- **手工业/商业分布图**
- **人口变化曲线图**
- **城市布局图**：长安城、汴京等古代城市平面图

### 1.5 文化与科技图形
- **文物/遗址图片**：青铜器、陶瓷、建筑等
- **科技发明示意图**：四大发明、天文仪器等
- **思想流派关系图**：百家争鸣、文艺复兴等
- **宗教传播路线图**

### 1.6 历史文献与史料
- **历史文献截图/摹本**：甲骨文、竹简、碑刻
- **历史照片**：近现代历史事件照片
- **漫画/宣传画**：政治漫画、战争宣传画
- **条约/文件示意图**：重要条约的签订场景或内容摘要

### 1.7 因果关系与比较图
- **历史事件因果链图**：原因→经过→结果→影响
- **对比表格图**：中外制度/事件对比
- **思维导图**：历史专题的知识结构

## 2. 物理可实现性约束

### 2.1 时间线约束
- 时间必须严格按照先后顺序排列（从左到右或从上到下）
- 公元前(BC)年份数值递减方向必须正确（如公元前221年在公元前206年左边）
- 时间跨度的间距应大致成比例（或明确标注为非等比）
- 朝代更替不得出现时间重叠（除并立政权外，如南宋与金）
- 并立政权必须在同一时间段内并行标注

### 2.2 历史地图约束
- 疆域边界必须与该历史时期的实际疆域一致
- 地名必须使用该历史时期的名称（如"长安"而非"西安"用于唐代）
- 河流、山脉等自然地理要素的位置必须正确
- 战争路线必须符合历史记载的进军方向
- 地图投影和比例尺必须合理

### 2.3 政治制度图约束
- 权力层级关系必须正确（上级在上，下级在下）
- 隶属关系必须与历史事实一致
- 同级机构必须在同一水平线上
- 制度演变的时间顺序必须正确

### 2.4 文物图片约束
- 文物图片必须与所述历史时期匹配
- 文物的材质、风格必须符合时代特征
- 不得使用后世仿品图片代替原物
- 建筑风格必须与时代一致（如唐代建筑不能出现明清风格的飞檐）

### 2.5 统计数据图约束
- 数据必须有可靠的历史来源
- 数据趋势必须与历史事实一致
- 图表类型必须适合所展示的数据

## 3. 几何关系约束

### 3.1 时间线布局
- 时间轴必须为直线（水平或垂直）
- 事件标注点必须精确对应时间位置
- 并行时间轴必须对齐相同的时间刻度
- 时间段用线段或色块表示，起止点必须准确

### 3.2 层级结构图布局
- 最高权力机构位于图的顶部
- 同级机构水平对齐
- 上下级之间用连线表示隶属关系
- 连线不得交叉（尽量避免）

### 3.3 地图几何约束
- 疆域边界线必须闭合
- 不同政权的疆域不得重叠（除争议地区外）
- 路线图的起点和终点必须标注
- 箭头方向必须与行进方向一致

### 3.4 因果关系图约束
- 因果链必须单向（从因到果）
- 多因一果或一因多果必须用分支表示
- 不得出现因果循环（除明确标注的历史循环论外）

---

## 4. 标注规范

### 4.1 时间标注
- 中国历史使用公元纪年，可附注朝代年号
- 公元前标注"公元前XXX年"或"前XXX年"
- 世纪标注格式："X世纪"或"X世纪XX年代"
- 朝代名称使用全称（如"东汉"而非"后汉"）

### 4.2 地图标注
- 国名/政权名使用该时期的正式名称
- 地名使用该时期的历史地名
- 战役名称标注在战场位置
- 路线用箭头标注方向，关键节点标注地名和日期

### 4.3 人物标注
- 人物姓名使用全名
- 可附注生卒年份
- 职务/身份标注在姓名下方
- 人物关系用连线标注

### 4.4 图表标注
- 数据来源必须标注
- 坐标轴必须标注名称和单位
- 图例必须完整
- 标题必须包含时间范围和主题

## 5. 常见错误清单

| 错误类型 | 错误描述 | 修正方案 |
|---------|---------|---------|
| 时间顺序错误 | 朝代顺序排列错误 | 严格按历史年表排序 |
| 公元前后混淆 | 公元前年份排列方向错误 | 公元前数值从大到小（左到右） |
| 疆域范围错误 | 唐代疆域包含了宋代才有的区域 | 按具体历史时期的疆域数据渲染 |
| 地名时代错误 | 唐代地图使用现代地名 | 使用该时期的历史地名 |
| 制度层级错误 | 三省六部的隶属关系画反 | 对照历史教材校验制度结构 |
| 文物时代错配 | 商代题目配了汉代文物图 | 文物图片必须与题目时代匹配 |
| 战争路线错误 | 进军方向与历史记载相反 | 对照史料校验路线方向 |
| 并立政权遗漏 | 南宋时期未标注金/蒙古 | 并立政权必须同时标注 |
| 数据趋势矛盾 | 工业革命时期GDP下降 | 数据趋势必须与历史事实一致 |
| 因果关系倒置 | 结果标注为原因 | 因果链方向必须正确 |
| 建筑风格穿越 | 秦代场景出现明清建筑 | 建筑风格必须与时代匹配 |
| 人物服饰错误 | 唐代人物穿清代服饰 | 服饰必须符合时代特征 |
