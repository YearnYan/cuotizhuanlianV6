# 政治 - 题目图形约束规范

## 1. 图形类型清单

### 1.1 政治制度与体制图
- **国家机构体系图**：全国人大、国务院、最高法/最高检等的关系
- **政党制度图**：中国共产党领导的多党合作制度
- **民族区域自治制度图**：自治区、自治州、自治县层级
- **基层群众自治制度图**：村委会、居委会
- **选举制度流程图**：直接选举/间接选举流程
- **国际组织结构图**：联合国、WTO等组织架构

### 1.2 经济学图形
- **供给需求曲线图**：供给曲线、需求曲线、均衡价格
- **价格变动影响图**：价格与需求量的关系
- **GDP/CPI等经济指标图**：柱状图、折线图
- **财政收支图**：国家财政收入与支出
- **货币流通示意图**
- **国际贸易流程图**：进出口贸易示意

### 1.3 哲学图形
- **唯物辩证法关系图**：对立统一、量变质变、否定之否定
- **认识论过程图**：实践→认识→再实践→再认识
- **矛盾关系图**：主要矛盾与次要矛盾、矛盾的主要方面与次要方面
- **联系与发展示意图**
- **意识与物质关系图**

### 1.4 文化与社会图形
- **文化传承与创新图**：传统文化→继承→创新
- **社会主义核心价值观图**：三个层面的关系
- **文化多样性示意图**
- **社会治理结构图**

### 1.5 法治图形
- **立法/执法/司法流程图**
- **公民权利与义务关系图**
- **法律体系层级图**：宪法→法律→行政法规→地方性法规
- **诉讼程序流程图**

### 1.6 时事与综合
- **新闻事件配图**：时政新闻的场景图
- **数据统计图表**：社会调查数据的可视化
- **漫画/讽刺画**：政治漫画类题目配图
- **思维导图**：知识点归纳的结构图

## 2. 物理可实现性约束

### 2.1 制度结构图约束
- 权力机关的层级关系必须正确：
  - 全国人民代表大会是最高国家权力机关
  - 国务院是最高国家行政机关
  - 最高人民法院是最高审判机关
  - 最高人民检察院是最高检察机关
- 领导关系与监督关系必须区分：
  - 人大与"一府一委两院"是产生、监督关系
  - 上下级政府是领导关系
  - 党的领导是全面领导
- 同级机关之间的关系必须正确标注

### 2.2 经济学图形约束
- **需求曲线**：必须向右下方倾斜（价格↑，需求量↓）
- **供给曲线**：必须向右上方倾斜（价格↑，供给量↑）
- **均衡价格**：必须位于供给曲线与需求曲线的交点
- 需求曲线移动：收入增加→右移，收入减少→左移
- 供给曲线移动：成本降低→右移，成本增加→左移
- 弹性大的曲线较平缓，弹性小的曲线较陡峭

### 2.3 哲学关系图约束
- 对立统一关系必须双向标注
- 因果关系必须单向（从因到果）
- 辩证关系不得简化为单向关系
- 矛盾的主要方面与次要方面必须区分标注
- 量变到质变必须有明确的度/临界点标注

### 2.4 法律层级约束
- 法律效力层级必须正确：宪法 > 法律 > 行政法规 > 地方性法规
- 立法主体必须与法律层级对应
- 权利与义务必须对应标注

### 2.5 数据图表约束
- 统计数据必须有来源依据
- 数据趋势必须与实际情况一致
- 图表类型必须适合数据特征
- 比例和刻度必须准确

## 3. 几何关系约束

### 3.1 层级结构图
- 最高权力机关位于图的最顶部
- 同级机关水平对齐
- 上下级关系用垂直连线表示
- 平级关系用水平连线表示
- 监督关系用虚线箭头，领导关系用实线箭头

### 3.2 经济学曲线约束
- 坐标轴必须垂直，刻度均匀
- 供给曲线和需求曲线的交点必须精确
- 曲线移动前后必须平行（平移）
- 消费者剩余和生产者剩余的阴影区域必须正确

### 3.3 流程图约束
- 流程方向必须一致（从上到下或从左到右）
- 判断节点必须有"是/否"两个分支
- 流程必须有明确的起点和终点
- 循环流程必须有退出条件

### 3.4 关系图约束
- 双向关系用双向箭头
- 单向关系用单向箭头
- 包含关系用嵌套框表示
- 并列关系的节点必须在同一层级

## 4. 标注规范

### 4.1 机构名称标注
- 使用官方全称或规范简称
- 全国人民代表大会（可简称"全国人大"）
- 中国人民政治协商会议（可简称"全国政协"）
- 国际组织使用官方中文名称和英文缩写

### 4.2 关系标注
- 领导关系：实线箭头 + "领导"
- 监督关系：虚线箭头 + "监督"
- 产生关系：实线箭头 + "产生/选举"
- 负责关系：实线箭头 + "负责/报告工作"

### 4.3 经济学标注
- 横轴标注：Q（数量）
- 纵轴标注：P（价格）
- 供给曲线标注：S（Supply）
- 需求曲线标注：D（Demand）
- 均衡点标注：E（Equilibrium）
- 移动后的曲线标注：S'、D'

### 4.4 数据标注
- 数据来源必须标注（如"数据来源：国家统计局"）
- 时间范围必须标注
- 单位必须标注（元、万元、亿元、%等）
- 图例必须完整

## 5. 常见错误清单

| 错误类型 | 错误描述 | 修正方案 |
|---------|---------|---------|
| 机构层级错误 | 国务院位于人大之上 | 人大是最高权力机关，位于最顶部 |
| 关系类型错误 | 人大与政府标注为"领导"关系 | 应为"产生、监督"关系 |
| 需求曲线方向错 | 需求曲线向右上方倾斜 | 需求曲线必须向右下方倾斜 |
| 供给曲线方向错 | 供给曲线向右下方倾斜 | 供给曲线必须向右上方倾斜 |
| 均衡点偏移 | 均衡点不在两曲线交点上 | 均衡点必须精确位于交点 |
| 法律层级错误 | 行政法规效力高于法律 | 严格按宪法>法律>行政法规排列 |
| 哲学关系简化 | 辩证关系画成单向箭头 | 辩证关系必须双向标注 |
| 矛盾主次不分 | 主要矛盾与次要矛盾未区分 | 用不同线条粗细或颜色区分 |
| 流程缺少终点 | 流程图没有结束节点 | 流程图必须有明确的终点 |
| 数据来源缺失 | 统计图表未标注数据来源 | 必须标注数据来源 |
| 机构名称错误 | 使用非官方简称 | 使用官方全称或规范简称 |
| 漫画寓意偏差 | 漫画内容与题目考查点不匹配 | 漫画必须准确反映题目考查的知识点 |
