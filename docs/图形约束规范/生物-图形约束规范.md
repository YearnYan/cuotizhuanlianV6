## 2. 物理可实现性约束（核心约束）

### 2.1 细胞结构约束

#### 2.1.1 细胞类型区分（核心约束）
- **动物细胞**：无细胞壁、无叶绿体、无大液泡、有中心体
- **植物细胞**：有细胞壁、有叶绿体（绿色部分）、有大液泡、高等植物无中心体
- **原核细胞**：无核膜包被的细胞核（拟核）、无线粒体/叶绿体/内质网/高尔基体、有核糖体
- 禁止在原核细胞中画出线粒体、叶绿体等膜性细胞器
- 禁止在动物细胞中画出细胞壁或叶绿体

#### 2.1.2 细胞器结构约束
- **线粒体**：双层膜结构，内膜向内折叠形成嵴
- **叶绿体**：双层膜结构，内部有基粒（类囊体堆叠）和基质
- **内质网**：单层膜，分为粗面（有核糖体附着）和滑面
- **高尔基体**：单层膜，扁平囊状结构堆叠
- **核糖体**：无膜结构，由大小亚基组成
- **细胞核**：双层核膜，有核孔，内含染色质和核仁

#### 2.1.3 细胞膜模型约束
- 磷脂双分子层：亲水头朝外，疏水尾朝内
- 蛋白质：有的贯穿磷脂双分子层，有的镶在表面，有的嵌入
- 糖蛋白只出现在细胞膜外侧
- 胆固醇嵌入磷脂双分子层中（动物细胞）

### 2.2 遗传学约束

#### 2.2.1 遗传图谱约束（核心约束）
- **符号规范**：
  - 男性用正方形 □，女性用圆形 ○
  - 患者用实心填充 ■ / ●
  - 携带者用半实心或中心加点
  - 婚配用横线连接 □—○
  - 后代用竖线从横线向下引出
- **遗传规律必须正确**：
  - 常染色体显性：患者的双亲中至少一方为患者
  - 常染色体隐性：患者的双亲可以都表现正常（携带者）
  - X染色体显性：患病父亲的女儿全部患病
  - X染色体隐性：患病母亲的儿子全部患病
  - 后代的基因型必须能从亲本基因型推导出来
- **禁止出现遗传学矛盾**：如两个正常父母生出显性遗传病患儿

#### 2.2.2 细胞分裂图约束
- 有丝分裂：DNA复制后染色体数目先不变（前期、中期），后期加倍，末期恢复
- 减数分裂I：同源染色体配对（联会）→分离
- 减数分裂II：姐妹染色单体分离
- 染色体数目变化必须与分裂时期一致
- 纺锤丝必须连接着丝点

### 2.3 生态学约束

#### 2.3.1 食物链/食物网约束（核心约束）
- **箭头方向必须正确**：从被捕食者指向捕食者（能量流动方向）
- **食物链必须以生产者开始**：第一营养级必须是植物/藻类等自养生物
- **分解者不参与食物链**：分解者单独标注，不在食物链箭头中
- **营养级不得超过5级**（通常4-5级）
- **食物网中不得出现环路**：A→B→C→A 是禁止的

#### 2.3.2 能量流动约束
- 能量传递效率约10%-20%
- 能量流动方向必须单向（低营养级→高营养级）
- 每个营养级的能量必须小于前一营养级
- 能量金字塔不得倒置

#### 2.3.3 种群增长曲线约束
- J型增长：指数增长，无环境阻力
- S型增长：有K值（环境容纳量），曲线在K值处趋于平稳
- K/2处增长速率最大
- 种群数量不得超过K值（长期稳定状态下）

## 3. 几何关系约束

### 3.1 细胞结构布局
- 细胞核通常位于细胞中央或偏中央位置
- 植物细胞的大液泡占据细胞大部分空间，将细胞质挤向边缘
- 细胞器大小比例必须合理：细胞核 > 线粒体 > 核糖体
- 细胞膜必须包围所有细胞内容物
- 细胞壁（植物）必须在细胞膜外侧

### 3.2 DNA结构几何约束
- DNA双螺旋必须为右手螺旋
- 碱基配对必须正确：A-T（两个氢键）、G-C（三个氢键）
- 两条链必须反向平行（5'→3' 和 3'→5'）
- 螺旋的螺距和直径必须均匀

### 3.3 遗传图谱布局
- 同一代的个体必须在同一水平线上
- 代际之间的垂直间距必须一致
- 婚配连线必须水平
- 后代连线必须垂直向下

### 3.4 生态系统图布局
- 食物链/食物网中营养级从低到高排列（通常从下到上或从左到右）
- 能量金字塔必须底宽顶窄
- 物质循环图中箭头必须形成完整循环

## 4. 标注规范

### 4.1 细胞结构标注
- 每个细胞器必须用引线标注名称
- 引线不得交叉
- 标注文字统一放在图形外侧
- 放大图必须标注放大倍数

### 4.2 遗传学标注
- 基因型使用斜体字母（如 Aa, XᴮXᵇ）
- 显性基因用大写，隐性基因用小写
- 系谱图中每个个体必须编号（如 I-1, I-2, II-1...）
- 世代用罗马数字标注（I, II, III...）

### 4.3 生态学标注
- 物种名称标注在对应图标旁
- 箭头上可标注能量传递效率
- 营养级编号标注（第一营养级、第二营养级...）
- 种群数量/密度的单位必须标注

### 4.4 生理学标注
- 器官/组织名称必须准确
- 血管标注动脉（红色）和静脉（蓝色）
- 神经信号传导方向用箭头标注
- 激素名称标注在分泌腺旁

## 5. 常见错误清单

| 错误类型 | 错误描述 | 修正方案 |
|---------|---------|---------|
| 细胞器错放 | 动物细胞中出现叶绿体 | 按细胞类型严格筛选细胞器 |
| 原核细胞有核膜 | 原核细胞画出了核膜包被的细胞核 | 原核细胞只有拟核，无核膜 |
| 磷脂方向错误 | 磷脂亲水头朝内 | 亲水头必须朝向水相（细胞内外） |
| 糖蛋白位置错误 | 糖蛋白出现在细胞膜内侧 | 糖蛋白只在细胞膜外侧 |
| 食物链箭头反向 | 箭头从捕食者指向被捕食者 | 箭头必须从被捕食者指向捕食者 |
| 食物链起点错误 | 食物链以动物开始 | 食物链必须以生产者开始 |
| 食物网有环路 | 出现A→B→C→A的循环 | 食物网不得出现环路 |
| 遗传图谱矛盾 | 两个正常父母生出显性遗传病患儿 | 校验后代基因型是否可从亲本推导 |
| 系谱符号错误 | 男性用圆形表示 | 男性□，女性○，严格遵守 |
| DNA碱基配对错误 | A与G配对 | A-T, G-C 严格配对 |
| DNA螺旋方向错误 | 画成左手螺旋 | DNA B型为右手螺旋 |
| 染色体数目错误 | 减数分裂I后期染色体数目不变 | 按分裂时期校验染色体数目变化 |
| 能量金字塔倒置 | 高营养级能量大于低营养级 | 能量金字塔不可倒置 |
| 种群曲线超过K值 | S型增长曲线最终超过K值 | S型曲线在K值处趋于稳定 |
| 反射弧方向错误 | 信号从效应器传向感受器 | 反射弧方向：感受器→传入神经→神经中枢→传出神经→效应器 |
| 血管颜色错误 | 动脉用蓝色表示 | 动脉红色，静脉蓝色 |
