# 英语 - 题目图形约束规范

## 1. 图形类型清单

### 1.1 语言学习类
- **四线三格**：英文字母书写练习（小学阶段）
- **字母卡片**：大小写字母展示
- **音标图**：国际音标表、口腔发音部位图
- **单词卡片**：图文对应的词汇卡

### 1.2 情景对话类
- **对话场景图**：购物、问路、就医等场景插图
- **人物对话气泡图**：带对话框的人物交互图
- **连环画/漫画**：故事情节的多格漫画
- **地图/路线图**：问路题的街道地图

### 1.3 阅读理解配图
- **文章主题插图**：与阅读材料相关的场景图
- **信息图表**：图表类阅读理解题的数据图
- **广告/海报/通知**：应用文阅读的版面图
- **时间表/日程表**：时刻表、课程表等表格图
- **菜单/价目表**：餐厅菜单、商品价格表

### 1.4 语法与写作类
- **句子结构图**：英语句子成分分析树
- **时态时间轴**：过去、现在、将来时态的时间线
- **思维导图**：写作提纲的结构图

### 1.5 听力题配图
- **选项图片**：听力选择题的A/B/C选项图
- **地图标注图**：听力中的位置标注
- **表格填空图**：听力信息记录表

### 1.6 文化知识类
- **国旗/地标图**：英语国家文化介绍
- **节日场景图**：圣诞节、感恩节等节日插图
- **世界地图**：英语国家分布图

## 2. 物理可实现性约束

### 2.1 四线三格约束
- 四条横线必须严格平行且等间距
- 英文字母必须书写在正确的格位：
  - 大写字母占中上两格
  - 小写字母如 a, c, e 占中格
  - 小写字母如 b, d, f 占中上格
  - 小写字母如 g, p, y 占中下格
- 字母间距必须均匀

### 2.2 场景图约束
- 场景中的物体比例必须合理（人物不能比建筑大）
- 透视关系必须一致（近大远小）
- 场景元素必须与题目描述一致
- 人物动作必须符合物理规律（站立时双脚着地）

### 2.3 地图/路线图约束
- 道路必须连通，不得出现断路
- 方向标注必须正确（北在上方为默认）
- 建筑物/地标位置必须与题目描述一致
- 比例尺（如有）必须一致

### 2.4 表格类约束
- 表格行列必须对齐
- 单元格大小应能容纳内容
- 表头必须清晰区分于数据行
- 时间表中的时间必须按顺序排列

---

## 3. 几何关系约束

### 3.1 对话气泡图
- 对话气泡的指向箭头必须指向说话者
- 对话顺序必须从上到下或从左到右
- 多人对话时，不同说话者的气泡应有明确区分

### 3.2 时间轴约束
- 时态时间轴必须从左（过去）到右（将来）
- 现在时刻必须明确标注
- 时间点标注必须与时态对应正确

### 3.3 信息图表约束
- 数据图表必须与文本中的数据一致
- 图表类型必须适合所展示的数据类型
- 图例必须完整且与图形对应

### 3.4 漫画/连环画约束
- 画格顺序必须从左到右、从上到下
- 同一人物在不同画格中的外观必须一致
- 时间线必须连贯，不得出现逻辑跳跃

---

## 4. 标注规范

### 4.1 文字标注
- 英文文字使用标准英文字体（Times New Roman / Arial）
- 中文提示语使用宋体/黑体
- 选项标注使用大写字母：A, B, C, D
- 图片编号格式：Picture 1 / Figure 1

### 4.2 对话标注
- 说话者姓名标注在对话气泡外侧或上方
- 对话文本使用引号或直接写在气泡内
- 省略号表示未完成的对话："..."

### 4.3 场景标注
- 地点名称使用英文标注（如 Library, Hospital）
- 方向标注使用英文（North, South, East, West）
- 价格标注使用对应货币符号（$, £, €）

### 4.4 音标标注
- 使用国际音标(IPA)标准符号
- 音标必须置于方括号 [ ] 或斜杠 / / 内
- 重音符号必须标注在正确的音节前

## 5. 常见错误清单

| 错误类型 | 错误描述 | 修正方案 |
|---------|---------|---------|
| 字母格位错误 | 小写g只占中格未延伸到下格 | 按字母规范校验每个字母的格位占用 |
| 对话顺序混乱 | 对话气泡排列顺序与对话逻辑不符 | 按对话轮次从上到下/从左到右排列 |
| 场景比例失调 | 人物与建筑物比例不合理 | 使用参考比例尺统一场景元素大小 |
| 地图方向错误 | 北方未朝上或方向标缺失 | 默认北方朝上，必须标注方向标 |
| 时间表顺序错 | 时间未按先后顺序排列 | 时间数据排序后再渲染 |
| 选项图片不清晰 | 听力选项图片辨识度低 | 使用简洁线条画，确保关键特征明显 |
| 文化元素错误 | 英国场景出现美国国旗 | 文化元素需与题目国家/地区匹配 |
| 音标符号错误 | 使用非标准音标符号 | 严格使用IPA标准符号库 |
| 价格符号错误 | 美元场景使用英镑符号 | 货币符号与题目语境匹配 |
| 广告版面混乱 | 广告/海报排版不符合英文排版习惯 | 参考真实英文广告版面设计 |
