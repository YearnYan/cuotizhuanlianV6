# 化学 - 题目图形约束规范

## 1. 图形类型清单

### 1.1 分子与原子结构图
- **原子结构示意图**：核外电子排布（圆圈+弧线+电子数）
- **离子结构示意图**：带电荷的原子结构
- **分子结构式**：用短线表示化学键的结构式
- **电子式**：用电子对表示化学键
- **球棍模型图**：原子用球表示，化学键用棍表示
- **空间填充模型图**：原子按比例紧密排列
- **晶体结构图**：NaCl、金刚石、石墨等晶体结构

### 1.2 化学方程式与反应图
- **化学方程式配图**：反应物→生成物的示意图
- **微观反应过程图**：分子拆分重组的微观示意
- **能量变化图**：反应过程中能量变化曲线（活化能图）
- **化学平衡图**：浓度-时间图、速率-时间图

### 1.3 实验装置图
- **气体制备装置**：制取O₂、H₂、CO₂等的装置
- **气体收集装置**：排水集气法、向上/向下排空气法
- **蒸馏装置**：蒸馏烧瓶、冷凝管、接收瓶
- **过滤装置**：铁架台、漏斗、滤纸、烧杯
- **滴定装置**：酸式/碱式滴定管、锥形瓶
- **电解装置**：电解槽、电极、电源
- **原电池装置**：两种电极、电解质溶液、导线

### 1.4 元素周期表相关
- **元素周期表（局部）**：展示特定区域的元素
- **元素性质变化趋势图**：原子半径、电负性等的变化曲线
- **电离能变化图**：逐级电离能的柱状图

### 1.5 有机化学图形
- **有机物结构简式**
- **同分异构体结构图**
- **有机反应机理图**：取代、加成、消去等反应过程
- **高分子链段结构图**

### 1.6 溶液与胶体
- **溶解度曲线**：温度-溶解度关系图
- **pH变化曲线**：滴定过程中pH变化
- **丁达尔效应示意图**

## 2. 物理可实现性约束（核心约束）

### 2.1 分子结构约束

#### 2.1.1 化学键规则
- **共价键数目必须正确**：
  - H: 1个共价键
  - O: 2个共价键
  - N: 3个共价键
  - C: 4个共价键
  - 卤素(F/Cl/Br/I): 1个共价键
- **离子键必须在金属与非金属之间**（或铵根与酸根之间）
- **配位键必须标注方向**（从提供孤对电子的原子指向接受的原子）
- **双键用两条平行线表示，三键用三条平行线表示**

#### 2.1.2 分子几何构型约束
- CH₄: 正四面体，键角109.5°
- CO₂: 直线形，键角180°
- H₂O: V形，键角约104.5°
- NH₃: 三角锥形，键角约107°
- BF₃: 平面三角形，键角120°
- 苯环: 正六边形，所有键角120°

#### 2.1.3 晶体结构约束
- NaCl晶体：每个Na⁺周围6个Cl⁻，每个Cl⁻周围6个Na⁺
- 金刚石：每个C原子连接4个C原子，正四面体排列
- 石墨：层内正六边形排列，层间范德华力
- 晶胞中原子计数必须正确（顶点1/8，棱1/4，面1/2，体心1）

### 2.2 实验装置约束

#### 2.2.1 气体收集装置方向约束（核心约束）
- **向上排空气法**：集气瓶正放（瓶口朝上），适用于密度大于空气的气体（CO₂、Cl₂、HCl等）
- **向下排空气法**：集气瓶倒放（瓶口朝下），适用于密度小于空气的气体（H₂、NH₃等）
- **排水集气法**：集气瓶装满水倒扣在水槽中，适用于不溶于水的气体
- 导管伸入集气瓶的深度必须正确：排空气法导管伸到瓶底，排水法导管刚伸入瓶口

#### 2.2.2 装置连接约束
- **装置连接必须密封**：橡皮管连接处不得有缝隙
- **导管方向必须正确**：气体发生装置的导管必须向下倾斜（防止冷凝水回流）
- **安全瓶/尾气处理必须存在**：有毒气体实验必须有尾气处理装置
- **加热装置位置正确**：酒精灯必须在需加热的容器下方
- **铁架台夹持位置正确**：夹在容器的中上部

#### 2.2.3 蒸馏装置约束
- 温度计水银球必须位于支管口处
- 冷凝水必须下进上出（逆流冷凝）
- 接收瓶不得密封（防止压力过大）

#### 2.2.4 滴定装置约束
- 酸式滴定管：玻璃活塞，用于酸性/氧化性溶液
- 碱式滴定管：橡皮管+玻璃珠，用于碱性溶液
- 滴定管刻度：0在上方，数值向下递增
- 读数时视线必须与液面凹面最低点平齐

### 2.3 反应条件约束
- 加热符号"△"必须标注在需要加热的反应中
- 催化剂必须标注在箭头上方
- 点燃条件必须标注"点燃"
- 电解条件必须标注"电解"
- 反应条件不得遗漏或错标

## 3. 几何关系约束

### 3.1 分子结构几何约束
- 球棍模型中原子球的大小必须反映原子半径的相对大小
- 化学键的长度应反映键长的相对大小（单键 > 双键 > 三键的键长关系）
- 空间构型的键角必须与理论值一致
- 苯环必须为正六边形，C-C键长相等

### 3.2 实验装置几何约束
- 铁架台必须竖直放置
- 酒精灯火焰必须朝上
- 试管倾斜角度必须与实验要求一致（加热固体时管口略向下倾斜）
- 量筒必须竖直放置
- 天平必须水平放置

### 3.3 图表几何约束
- 溶解度曲线的坐标轴必须正确（横轴温度，纵轴溶解度）
- 能量变化图中反应物和生成物的能量高低必须正确
- pH曲线中酸碱滴定的突变区域必须明显

## 4. 标注规范

### 4.1 化学符号标注
- 元素符号必须使用标准写法（首字母大写，第二字母小写）
- 化学式中的下标数字必须为下标格式
- 离子电荷标注在右上角（如 Na⁺, Cl⁻, SO₄²⁻）
- 化合价标注在元素正上方（如 +1价标为 $\overset{+1}{Na}$）

### 4.2 实验装置标注
- 每个仪器必须标注名称
- 药品名称/化学式必须标注在对应容器旁
- 气体流向必须用箭头标注
- 加热位置必须标注火焰符号
- 夹持位置必须明确

### 4.3 图表标注
- 坐标轴必须标注物理量和单位
- 曲线必须标注对应的物质或条件
- 关键数据点必须标注数值
- 图例必须完整

### 4.4 结构式标注
- 共价键用短线"—"表示
- 双键用"="表示
- 三键用"≡"表示
- 配位键用"→"表示
- 孤对电子用":"表示

## 5. 常见错误清单

| 错误类型 | 错误描述 | 修正方案 |
|---------|---------|---------|
| 化学键数目错误 | 碳原子只连了3个键 | 校验每个原子的成键数是否符合价键规则 |
| 分子构型错误 | CH₄画成平面正方形 | 按VSEPR理论确定正确的空间构型 |
| 气体收集方向错误 | 用向上排空气法收集H₂ | 根据气体密度选择正确的收集方法 |
| 导管方向错误 | 制气装置导管向上倾斜 | 导管必须向下倾斜防止冷凝水回流 |
| 冷凝水方向错误 | 冷凝水上进下出 | 冷凝水必须下进上出（逆流冷凝） |
| 温度计位置错误 | 蒸馏温度计插入液面以下 | 水银球必须位于支管口处 |
| 滴定管刻度反向 | 0刻度在下方 | 滴定管0刻度在上方 |
| 晶胞原子计数错误 | NaCl晶胞中Na⁺数目计算错误 | 按位置系数（顶1/8，棱1/4，面1/2，体1）计算 |
| 苯环变形 | 苯环画成不规则六边形 | 苯环必须为正六边形 |
| 电极标注错误 | 电解池正负极标反 | 电解池：阳极接正极，阴极接负极 |
| 装置密封缺失 | 有毒气体实验无尾气处理 | 必须添加尾气处理装置 |
| 能量图高低错误 | 放热反应中生成物能量高于反应物 | 放热反应生成物能量必须低于反应物 |
