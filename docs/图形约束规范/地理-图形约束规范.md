# 地理 - 题目图形约束规范

## 1. 图形类型清单

### 1.1 地图类
- **世界地图/区域地图**：政区图、地形图
- **中国地图**：省级行政区划图、地形图
- **等高线地形图**：山峰、山谷、山脊、山脚、鞍部、陡崖
- **地形剖面图**：沿某条线的地形起伏
- **专题地图**：气候分布图、植被分布图、人口分布图、资源分布图
- **交通/路线图**：铁路、公路、航线

### 1.2 天文与地球运动图形
- **地球公转示意图**：椭圆轨道、近日点/远日点
- **地球自转示意图**：自转方向、昼夜交替
- **太阳直射点移动图**：回归运动示意
- **日照图/光照图**：侧视图、俯视图（极地投影）
- **太阳高度角示意图**
- **时区图**：24个时区的划分

### 1.3 大气与气候图形
- **大气环流图**：三圈环流、季风环流
- **气压带风带图**：全球气压带和风带分布
- **气旋/反气旋图**：低压/高压系统的气流运动
- **锋面示意图**：冷锋、暖锋、准静止锋
- **气温/降水柱状图**：各月气温曲线和降水柱状图
- **气候类型分布图**

### 1.4 水文图形
- **水循环示意图**：蒸发、降水、径流
- **洋流分布图**：全球洋流模式
- **河流水系图**：干流、支流、流域
- **河流流量过程线图**

### 1.5 地质图形
- **地球内部结构图**：地壳、地幔、地核
- **板块构造图**：六大板块分布及运动方向
- **地质构造图**：背斜、向斜、断层
- **岩石圈物质循环图**：三大类岩石的转化
- **地质剖面图**：岩层分布与构造

### 1.6 人文地理图形
- **人口金字塔图**
- **城市功能分区图**：同心圆、扇形、多核心模式
- **工业/农业分布图**
- **交通网络图**
- **城市化进程曲线图**

### 1.7 统计图表
- **柱状图/折线图/饼图**：地理数据统计
- **坐标图**：双Y轴图（气温+降水）
- **三角坐标图**：人口年龄结构、产业结构

## 2. 物理可实现性约束

### 2.1 地图约束

#### 2.1.1 地图三要素
- **比例尺**：必须标注，线段式/数字式/文字式
- **方向标**：默认"上北下南左西右东"，非默认必须标注指北针
- **图例**：所有符号必须在图例中解释

#### 2.1.2 等高线约束
- 等高线必须闭合（可在图幅边缘截断）
- 等高距必须一致（同一幅图中相邻等高线的高差相等）
- 等高线不得相交（除陡崖外）
- 陡崖处等高线重合
- 等高线密集处坡度陡，稀疏处坡度缓
- 山脊：等高线向低处凸出
- 山谷：等高线向高处凸出
- 鞍部：两组闭合等高线之间的区域

#### 2.1.3 经纬线约束
- 经线为南北方向的半圆弧，连接南北极
- 纬线为东西方向的完整圆，与赤道平行
- 经度范围：0°-180°E/W
- 纬度范围：0°-90°N/S
- 经纬线间距必须均匀（等间距投影下）

### 2.2 天文地理约束

#### 2.2.1 地球运动约束
- 地球自转方向：自西向东（从北极上空看逆时针，从南极上空看顺时针）
- 地球公转方向：自西向东（逆时针，从北极上空看）
- 地轴倾斜角度：约23.5°（黄赤交角）
- 太阳直射点范围：南北回归线之间（23.5°N - 23.5°S）

#### 2.2.2 日照图约束
- 晨昏线必须为大圆的一半，与太阳光线垂直
- 昼半球和夜半球面积必须相等
- 春秋分时晨昏线必须过南北极点
- 夏至时北极圈内全部为极昼，南极圈内全部为极夜
- 冬至时相反

### 2.3 大气约束

#### 2.3.1 气压带风带约束
- 赤道低压带位于赤道附近
- 副热带高压带位于南北纬30°附近
- 副极地低压带位于南北纬60°附近
- 极地高压带位于南北极
- 风向必须受地转偏向力影响：北半球右偏，南半球左偏

#### 2.3.2 气旋/反气旋约束
- 北半球气旋：逆时针辐合
- 北半球反气旋：顺时针辐散
- 南半球气旋：顺时针辐合
- 南半球反气旋：逆时针辐散
- 气旋中心为低压，反气旋中心为高压

#### 2.3.3 锋面约束
- 冷锋：冷气团主动推进，锋面符号为三角形（指向暖气团方向）
- 暖锋：暖气团主动推进，锋面符号为半圆形（指向冷气团方向）
- 冷锋坡度较陡，暖锋坡度较缓
- 降水区域：冷锋在锋后，暖锋在锋前

### 2.4 水文约束
- 洋流方向必须与实际一致（如北半球中低纬度顺时针环流）
- 暖流用红色箭头，寒流用蓝色箭头
- 河流必须从高处流向低处
- 河流流向必须与等高线垂直（在等高线地形图中）

## 3. 几何关系约束

### 3.1 地图投影约束
- 墨卡托投影：经纬线为互相垂直的直线，高纬度地区面积变形大
- 极地投影：经线为放射状直线，纬线为同心圆
- 等面积投影：面积比例正确，形状可能变形
- 同一幅图中投影方式必须一致

### 3.2 等高线几何约束
- 相邻等高线之间的高差等于等高距
- 等高线的弯曲方向指示地形特征
- 等高线间距（水平距离）与坡度成反比
- 山顶等高线为闭合曲线，中心标注高程

### 3.3 日照图几何约束
- 侧视图：地球为圆形，晨昏线为直线或弧线
- 俯视图（极地投影）：地球为圆形，经线为放射线，纬线为同心圆
- 太阳光线必须平行
- 晨昏线与太阳光线必须垂直

### 3.4 统计图几何约束
- 气温降水图：气温用折线（左Y轴），降水用柱状（右Y轴）
- 三角坐标图：三个轴的读数之和必须为100%
- 人口金字塔：左侧为男性，右侧为女性，中间为年龄段

## 4. 标注规范

### 4.1 地图标注
- 地名使用规范汉字，国际地图使用英文
- 河流名称沿河流走向标注
- 山脉名称沿山脉走向标注
- 城市用不同大小的圆点表示等级
- 国界用特定线型（如点划线）
- 省界用另一种线型

### 4.2 等高线标注
- 高程数值标注在等高线上，字头朝向高处
- 计曲线（每隔几条加粗的等高线）必须标注高程
- 特殊地形标注：山峰标注"▲"和高程，鞍部标注"×"

### 4.3 气候图标注
- 月份标注在横轴（1-12月）
- 气温单位：°C，降水单位：mm
- 气候类型名称标注在图表标题中
- 风向用风向标或箭头标注

### 4.4 方向与比例尺标注
- 指北针箭头必须指向正北
- 比例尺格式统一：1:50000 或 图上1cm代表实地500m
- 经纬度标注格式：XXX°E/W, XX°N/S

## 5. 常见错误清单

| 错误类型 | 错误描述 | 修正方案 |
|---------|---------|---------|
| 等高线相交 | 两条不同高程的等高线交叉 | 等高线不得相交（陡崖除外） |
| 等高距不一致 | 同一图中等高距忽大忽小 | 统一等高距，严格等差 |
| 地球自转方向错 | 从北极看顺时针旋转 | 北极俯视必须逆时针 |
| 晨昏线位置错 | 春分日晨昏线不过极点 | 春秋分晨昏线必须过南北极点 |
| 气旋旋转方向错 | 北半球气旋顺时针 | 北半球气旋逆时针辐合 |
| 风向偏转错误 | 北半球风向左偏 | 北半球右偏，南半球左偏 |
| 冷暖锋符号反 | 冷锋用半圆形符号 | 冷锋三角形，暖锋半圆形 |
| 洋流方向错误 | 暖流方向画反 | 对照实际洋流图校验 |
| 河流逆流 | 河流从低处流向高处 | 河流必须从高处流向低处 |
| 经纬度标注错 | 经度超过180° | 经度0°-180°，纬度0°-90° |
| 比例尺缺失 | 地图未标注比例尺 | 地图三要素不可缺少 |
| 三角坐标读数错 | 三个数值之和不等于100% | 三角坐标三轴之和必须为100% |
| 人口金字塔左右反 | 男性在右侧 | 男性左侧，女性右侧 |
| 太阳直射超范围 | 太阳直射点超过回归线 | 直射点范围23.5°N-23.5°S |
