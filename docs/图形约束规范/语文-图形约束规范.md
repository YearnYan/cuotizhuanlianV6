# 语文 - 题目图形约束规范

## 1. 图形类型清单

### 1.1 文字排版类
- **田字格**：用于汉字书写练习，小学低年级常见
- **米字格**：用于汉字笔画练习
- **四线三格**：用于拼音书写
- **方格稿纸**：用于作文书写示意

### 1.2 语言知识结构图
- **汉字结构分解图**：左右结构、上下结构、包围结构等
- **偏旁部首归类图**：偏旁与例字的对应关系
- **笔顺示意图**：汉字书写笔顺的分步展示
- **拼音声调示意图**：四声调值的折线示意

### 1.3 阅读理解类
- **文章结构图**：总分总、并列、递进等结构示意
- **思维导图/概念图**：文章主题与要点的层级关系
- **人物关系图**：小说/戏剧中人物关系网络
- **情节发展曲线图**：故事情节的起承转合示意
- **时间线/年表**：文学史、作者生平时间轴

### 1.4 修辞与语法类
- **句子成分分析图**：主谓宾定状补的层次划分线
- **复句关系图**：关联词与分句的逻辑关系
- **修辞手法示意图**：比喻、拟人等修辞的图示说明

### 1.5 古诗文配图
- **诗词意境插图**：山水、田园、边塞等场景
- **文言文情境图**：历史场景还原
- **书法作品图**：碑帖、名家书法展示

### 1.6 综合性学习
- **调查统计图表**：柱状图、饼图（语文综合实践活动）
- **手抄报版面设计图**：版面布局示意
- **宣传海报示意图**：活动宣传类图形

## 2. 物理可实现性约束

### 2.1 田字格/米字格约束
- 外框必须为正方形，边长相等
- 田字格内部十字线必须精确交于中心点
- 米字格对角线必须连接对角顶点
- 格子间距必须均匀一致
- 线条粗细：外框线 > 内部辅助线（虚线）

### 2.2 四线三格约束
- 四条横线必须等间距且严格平行
- 上格:中格:下格的高度比约为 1:1:1 或按教材标准 1:1:0.5
- 拼音字母必须书写在正确的格位内（如 b 占中上格，p 占中下格）

### 2.3 汉字结构图约束
- 汉字部件拆分必须符合国家语委《汉字部件规范》
- 结构比例必须反映真实书写比例（如"湖"左窄右宽）
- 笔顺编号必须符合《现代汉语通用字笔顺规范》

### 2.4 文章结构图约束
- 层级关系必须清晰，不得出现逻辑环路
- 并列关系的节点必须处于同一层级
- 包含关系不得交叉（A包含B，则B不能同时包含A）

### 2.5 人物关系图约束
- 关系连线必须双向标注（如"父子"关系两端分别标"父""子"）
- 同一人物不得出现多个节点
- 关系类型必须与原文一致，不得臆造

## 3. 几何关系约束

### 3.1 结构图布局
- 树形结构图：父节点居中于子节点上方，连线不得交叉
- 网络关系图：节点间连线应尽量避免交叉，交叉处需明确区分
- 流程图：箭头方向必须一致（从上到下或从左到右）

### 3.2 时间线约束
- 时间轴必须单向递增（从左到右或从上到下）
- 时间刻度间距应与实际时间跨度成正比（或明确标注为非等比）
- 事件标注点必须对齐到时间轴上的正确位置

### 3.3 统计图表约束
- 柱状图柱体宽度必须一致
- 饼图各扇区角度之和必须为360°
- 数据标注必须与图形比例一致

---

## 4. 标注规范

### 4.1 文字标注
- 字体：正文使用宋体/楷体，标题可用黑体
- 字号：标注文字不小于正文字号的80%
- 颜色：黑色为主，重点标注可用红色，辅助信息用灰色
- 拼音标注必须使用规范的拼音字体，声调标在主元音上

### 4.2 符号标注
- 笔顺编号使用阿拉伯数字，置于笔画起始端
- 结构分析使用标准符号："|"表示左右分割，"—"表示上下分割
- 句子成分分析使用统一的下划线标注体系：
  - 主语：双下划线 ═══
  - 谓语：单下划线 ───
  - 宾语：波浪线 ～～～
  - 定语：( )
  - 状语：[ ]
  - 补语：< >

### 4.3 图文配合
- 插图与题目文字的对应关系必须明确
- 图片编号格式统一："图X"或"图X-X"
- 图注位于图片正下方，居中排列

## 5. 常见错误清单

| 错误类型 | 错误描述 | 修正方案 |
|---------|---------|---------|
| 田字格变形 | 外框不是正方形，呈长方形 | 严格约束宽高比为1:1 |
| 米字格对角线偏移 | 对角线未通过中心点 | 对角线端点必须为格子四角 |
| 笔顺错误 | 笔顺编号与国家标准不符 | 对照《笔顺规范》逐字校验 |
| 拼音声调位置错误 | 声调标在错误的元音上 | 按"a-o-e-i-u-ü"优先级标调 |
| 四线三格比例失调 | 三格高度不均匀 | 使用等分算法确保比例一致 |
| 结构图层级混乱 | 并列关系的节点不在同一水平线 | 同级节点Y坐标强制对齐 |
| 人物关系矛盾 | 同一对人物标注了矛盾的关系 | 关系图需做一致性校验 |
| 时间线逆序 | 时间轴上后发事件在前 | 按时间戳排序后再渲染 |
| 汉字结构拆分错误 | 部件拆分不符合规范 | 引用标准部件库进行校验 |
| 句子成分标注重叠 | 多个成分标注线相互覆盖 | 分层标注，使用不同垂直偏移 |
| 古诗配图时代错误 | 唐诗配图出现明清建筑 | 配图需校验历史时代一致性 |
| 统计图数据不一致 | 图形比例与标注数值不匹配 | 数据驱动渲染，禁止手动调整图形 |
