# 防抄袭保护配置

## 生产环境配置

### 1. 关闭 Source Map

在生产环境中关闭source map，防止源代码泄露。

**package.json 添加构建脚本：**
```json
{
  "scripts": {
    "build": "NODE_ENV=production webpack --mode production",
    "build:prod": "cross-env NODE_ENV=production GENERATE_SOURCEMAP=false npm run build"
  }
}
```

### 2. 代码混淆

使用 Terser 进行代码混淆和压缩。

**安装依赖：**
```bash
npm install --save-dev terser-webpack-plugin
```

**webpack.config.js 配置：**
```javascript
const TerserPlugin = require('terser-webpack-plugin');

module.exports = {
  mode: 'production',
  optimization: {
    minimize: true,
    minimizer: [
      new TerserPlugin({
        terserOptions: {
          compress: {
            drop_console: true,
            drop_debugger: true,
            pure_funcs: ['console.log']
          },
          mangle: {
            safari10: true
          },
          output: {
            comments: false,
            ascii_only: true
          }
        },
        extractComments: false
      })
    ]
  },
  devtool: false // 关闭source map
};
```

### 4. API认证保护

**实现JWT认证中间件：**

创建 `server/middleware/auth.js`：
```javascript
const jwt = require('jsonwebtoken');

const JWT_SECRET = process.env.JWT_SECRET || 'your-secret-key-change-in-production';

function authMiddleware(req, res, next) {
  const token = req.headers.authorization?.replace('Bearer ', '');

  if (!token) {
    return res.status(401).json({ error: '未授权访问' });
  }

  try {
    const decoded = jwt.verify(token, JWT_SECRET);
    req.user = decoded;
    next();
  } catch (error) {
    return res.status(401).json({ error: '无效的访问令牌' });
  }
}

module.exports = { authMiddleware, JWT_SECRET };
```

**在路由中使用：**
```javascript
const { authMiddleware } = require('../middleware/auth');

// 保护敏感API
router.post('/exam/generate', authMiddleware, async (req, res) => {
  // 生成试卷逻辑
});
```

### 5. 请求频率限制

**安装依赖：**
```bash
npm install express-rate-limit
```

**配置限流：**
```javascript
const rateLimit = require('express-rate-limit');

const limiter = rateLimit({
  windowMs: 15 * 60 * 1000, // 15分钟
  max: 100, // 最多100个请求
  message: '请求过于频繁，请稍后再试'
});

app.use('/api/', limiter);
```

### 6. 环境变量保护

**创建 `.env` 文件（不提交到Git）：**
```env
NODE_ENV=production
PORT=5000
JWT_SECRET=your-super-secret-jwt-key-here
AI_API_KEY=your-ai-api-key-here
REDIS_URL=redis://localhost:6379
```

**创建 `.env.example` 文件（提交到Git）：**
```env
NODE_ENV=development
PORT=5000
JWT_SECRET=change-me-in-production
AI_API_KEY=your-api-key
REDIS_URL=redis://localhost:6379
```

**在 `.gitignore` 中添加：**
```
.env
.env.local
.env.production
node_modules/
dist/
build/
*.log
```

### 7. HTTPS强制

**生产环境强制HTTPS：**
```javascript
app.use((req, res, next) => {
  if (process.env.NODE_ENV === 'production' && !req.secure) {
    return res.redirect('https://' + req.headers.host + req.url);
  }
  next();
});
```

### 8. CORS配置

**限制允许的域名：**
```javascript
const cors = require('cors');

const corsOptions = {
  origin: process.env.NODE_ENV === 'production'
    ? ['https://yourdomain.com', 'https://www.yourdomain.com']
    : '*',
  credentials: true
};

app.use(cors(corsOptions));
```

4. **备份数据**：定期备份知识库和用户数据
