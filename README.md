# 全息试卷工厂 - 赛博出卷机 v11

AI驱动的智能K12试卷生成系统，支持多教材版本、多科目、多难度层级的原创试题生成。

## 核心特性

### 1. 多教材版本支持
- 人教版、苏教版、北师大版、鲁教版
- 覆盖小学1-6年级、初中1-3年级、高中1-3年级
- 9大科目：语文、数学、英语、物理、化学、生物、地理、历史、政治

### 2. 智能难度系统
- 10级难度层级，从基础识记到竞赛难题
- 每个难度级别有明确定义和区分度
- 可视化难度指示器

### 3. 题目选择组卷
- AI生成多道备选题目
- 用户可选择满意的题目进行组卷
- 支持按题型分组展示

### 4. LaTeX渲染架构
- 图文分离策略：简单公式前端渲染，复杂图形后端渲染
- 支持数学公式、化学方程式、物理图形等
- 使用KaTeX进行前端快速渲染

### 5. 导出功能
- 支持PDF导出（高质量打印）
- 支持Word文档导出
- 标准A4试卷格式，含密封线

## 技术架构

### 前端
- 纯HTML/CSS/JavaScript
- KaTeX数学公式渲染
- 响应式设计，支持桌面和平板

### 后端
- Node.js + Express
- RESTful API设计
- 知识库JSON存储

## 快速开始

### 安装依赖
```bash
npm install
```

### 启动服务器
```bash
npm start
```

服务器将在 http://localhost:5000 启动

### 开发模式
```bash
npm run dev
```

## API接口

### 知识点查询
```
GET /api/knowledge/topics?version=人教版&grade=高中一年级&subject=数学
```

### 生成试卷
```
POST /api/exam/generate
Body: {
  version: "人教版",
  grade: "高中一年级",
  subject: "数学",
  topic: "函数与导数",
  examPoint: "导数的应用",
  difficulty: 5,
  examType: "final"
}
```

### LaTeX渲染
```
POST /api/render/latex
Body: {
  latex: "\\frac{a}{b}",
  format: "svg",
  scale: 1
}
```

## 使用流程

1. **选择配置**
   - 选择教材版本（人教版/苏教版/北师大版/鲁教版）
   - 选择科目和年级
   - 选择知识点和考点

2. **设置参数**
   - 调整难度等级（1-10级）
   - 选择试卷类型（随堂小测/单元测试/期中/期末/模拟）
   - 决定是否显示答案

3. **生成题目**
   - 点击"生成题目"按钮
   - AI生成多道备选题目
   - 题目按题型分组展示

4. **选择组卷**
   - 浏览生成的题目
   - 点击选择满意的题目
   - 点击"组卷"按钮生成试卷

5. **导出试卷**
   - 预览试卷效果
   - 导出为PDF或Word格式
   - 打印或分享
